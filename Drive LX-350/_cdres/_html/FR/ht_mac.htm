Suivez la procédure ci-après pour ajouter le pilote d'imprimante à l'ordinateur. Si votre produit intègre une fonction de télécopie et que vous souhaitez envoyer des télécopies depuis votre ordinateur, n'oubliez pas d'ajouter le pilote de télécopie.
Sous Mac OS X 10.5 ou version ultérieure
Sous Mac OS X 10.4.x
Remarque :
Il n'est pas nécessaire d'ajouter à nouveau le ou les pilotes une fois qu'ils ont été ajoutés à moins de changer de méthode de connexion ou d'utiliser un autre produit.
Vérifiez que le produit est sous tension et correctement connecté à votre ordinateur à l'aide d'une connexion USB ou réseau.

Sous Mac OS X 10.5 ou version ultérieure

Concernant les connexions USB, le pilote d'imprimante est automatiquement ajouté et apparaît dans la fenêtre Print & Fax (Imprimantes et fax). Vous ne devez ajouter le pilote d'imprimante que si ce dernier n'apparaît pas ou si vous connectez l'imprimante via un réseau. Le pilote de télécopie (pour les modèles intégrant une fonction de télécopie) n'est pas ajouté automatiquement. Ajoutez le pilote de télécopie en même temps que le pilote d'imprimante.
Cliquez sur le bouton + (ajouter).
Cliquez sur Default (Par défaut) (ou Default Browser (Navigateur par défaut)). Sélectionnez votre produit Epson dans la liste, puis cliquez sur Add (Ajouter). Vérifiez dans le tableau ci-dessous que vous avez sélectionné le bon produit.

* XXXXXX = nom du produit, YYYYYY = 6 derniers chiffres de l'adresse MAC
La recherche du pilote de télécopie peut prendre quelques instants. Attendez que le pilote de télécopie apparaisse dans la liste, puis ajoutez-le en suivant la même procédure.
Sélectionnez Quit System Preferences (Quitter Préférences Système) dans le menu System Preferences (Préférences Système).

Sous Mac OS X 10.4.x

Pour ajouter le pilote d'imprimante, suivez la même procédure que sous Mac OS X 10.5 ou version ultérieure.
Pour ajouter le pilote de télécopie, procédez comme suit.
Cliquez sur Default Browser (Navigateur par défaut), puis sur le bouton More Printers (Plus d'imprimantes).
Dans le menu déroulant Printer Browser (Choix de l'imprimante), sélectionnez EPSON FAX. Attendez que le pilote de télécopie (FAX XXXXXX) apparaisse dans la liste. Lorsque le pilote de télécopie apparaît, sélectionnez-le puis cliquez sur Add (Ajouter).
Sélectionnez Quit System Preferences (Quitter Préférences Système) dans le menu System Preferences (Préférences Système).
