Басып шығарудан бұрын өніміңізге принтер драйверін қосуыңыз керек. Принтер драйверін қосу үшін төмендегі қадамдарды орындаңыз.
Mac OS X 10.5 немесе одан кейінгі нұсқалары үшін
Mac OS X 10.3-тен 10.4-ке дейінгі нұсқалары үшін
Ескертпе:
Принтер драйверін қосқаннан кейін, принтер драйверлерін ауыстырмасаңыз оны қайтадан қосуыңыздың қажеті жоқ.
USB кабелі арқылы принтерді компьютеріңізге мықтап жалғағаныңызға көз жеткізіңіз де, принтерді қосыңыз.

Mac OS X 10.5 немесе одан кейінгі нұсқалары үшін

+ қосу түймешігін басыңыз.
Default түймешігін басып, содан кейін USB қосылымы көрсетілген принтерді таңдаңыз. Print Using мәзірінде тиісті EPSON принтер драйвері көрсетілгеніне көз жеткізіңіз де, содан кейін Printer Browser терезесінде Add түймешігін басыңыз.
System Preferences мәзірінен Quit System Preferences пәрменін таңдаңыз.

Mac OS X 10.3-тен 10.4-ке дейінгі нұсқалары үшін

Print Center немесе Printer Setup Utility терезесінде Add түймешігін басыңыз.
Тиісті порт пен принтерді таңдау үшін амалдық жүйеңіздің нұсқауларын орындаңыз.

Mac OS X 10.3 нұсқасы үшін:

Порт ретінде USB портын таңдаңыз. Одан кейін принтерді таңдап, Printer List тілқатысу терезесінде Add түймешігін басыңыз.

Mac OS X 10,4 нұсқасы үшін:

Default Browser түймешігін басып, USB қосылымы көрсетілген принтерді таңдаңыз, содан кейін Printer Browser тілқатысу терезесінде Add түймешігін басыңыз.
Printer Setup Utility мәзірінен Quit Printer Setup Utility дегенді таңдаңыз.
