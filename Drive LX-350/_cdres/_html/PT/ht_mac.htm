Execute o processo seguinte para adicionar o controlador da impressora ao computador. Se o produto estiver equipado com uma função de fax e se pretender enviar faxes a partir do computador, assegure-se de que também adiciona o controlador do fax.

Para Mac OS X 10.4.x
Nota:
Após adicionar o(s) controlador(es), não é necessário adicioná-lo(s) novamente a não ser que altere o método de ligação ou utilize um produto diferente.
Assegure-se de que o produto está em funcionamento e que se encontra correctamente ligado ao computador através de uma ligação USB ou de rede.

Para Mac OS X 10.5 ou posterior

Para ligações USB, o controlador da impressora é automaticamente adicionado e apresentado na janela Print & Fax (Impressão e Fax). Tem apenas de adicionar o controlador da impressora se este não for apresentado ou se estiver a ligar a impressora através de uma rede. O controlador do fax (para modelos com uma função de fax) não é adicionado automaticamente. Adicione o controlador do fax assim como o controlador da impressora.
Clique no botão + para adicionar.
Clique em Default (Padrão) (ou Default Browser (Navegador Padrão)). Seleccione o produto Epson na lista e, em seguida, clique em Add (Adicionar). Consulte a tabela seguinte para confirmar se seleccionou o correcto.

Poderá ser necessário algum tempo para procurar o controlador do fax. Aguarde até que o controlador do fax seja apresentado na lista e, em seguida, adicione-o utilizando o mesmo processo.
Seleccione Quit System Preferences (Encerrar Preferências de Sistema) no menu System Preferences (Preferências do Sistema).

Para Mac OS X 10.4.x

Para adicionar o controlador da impressora, execute o mesmo processo que para Mac OS X 10.5 ou posterior.
Para adicionar o controlador do fax, execute o processo seguinte.
Clique em Default Browser (Navegador Padrão) e, em seguida, clique no botão More Printers (Mais Impressoras).
No menu pendente Printer Browser (Navegador Impressoras), seleccione EPSON FAX. Aguarde até que o controlador do fax (FAX XXXXXX) apareça na lista. Quando aparecer, seleccione o controlador e clique em Add (Adicionar).
Seleccione Quit System Preferences (Encerrar Preferências de Sistema) no menu System Preferences (Preferências do Sistema).
