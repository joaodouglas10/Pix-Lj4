A készülék használatához telepíteni kell a hozzá való nyomtató-illesztőprogramot. A nyomtató-illesztőprogram telepítéséhez járjon el az alábbiak szerint.
Mac OS X 10.5 vagy újabb esetében
Mac OS X 10.3–10.4 esetében
Megjegyzés:
A nyomtató-illesztőprogramot a telepítése után csak a nyomtató-illesztőprogram cseréje esetén kell újra a rendszerhez adni.
Csatlakoztassa a nyomtatót a számítógéphez az USB-kábellel, majd kapcsolja be a nyomtatót.

Mac OS X 10.5 vagy újabb esetében

Kattintson a + (hozzáadás) gombra.
Kattintson a Default lehetőségre, majd válassza ki azt a nyomtatót, amelynek az USB-kapcsolata látható. Ellenőrizze, hogy a Print Using menüben a megfelelő EPSON nyomtató-illesztőprogram jelenik-e meg, majd kattintson az Add gombra a Printer Browser ablakban.
Válassza a Quit System Preferences lehetőséget a System Preferences menüben.

Mac OS X 10.3–10.4 esetében

Kattintson az Add gombra a Print Center vagy a Printer Setup Utility képernyőn.
Kövesse az operációs rendszer utasításait, és válassza ki a megfelelő portot és nyomtatót.

Mac OS X 10.3 esetében:

A portválasztásnál válassza az USB lehetőséget. Utána jelölje ki a nyomtatót, és kattintson az Add gombra a Printer List párbeszédpanelen.

Mac OS X 10.4 esetében:

Kattintson a Default Browser lehetőségre, majd válassza ki azt a nyomtatót, amelynek az USB-kapcsolata látható, majd kattintson az Add gombra a Printer Browser párbeszédpanelen.
Válassza a Quit Printer Setup Utility lehetőséget a Printer Setup Utility menüben.
