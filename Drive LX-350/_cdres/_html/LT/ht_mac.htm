Prieš spausdinimą būtina pridėti jūsų spausdintuvo tvarkyklę. Norėdami pridėti spausdintuvo tvarkyklę atlikite žemiau nurodytus veiksmus.
„Mac“ OS X 10.5 ar vėlesnė versija
„Mac“ OS X 10.3 ir 10.4 versijos
Pastaba:
Po to, kai pridėjote spausdintuvo tvarkyklę, jums nereikia jos pridėti iš naujo, jei nekeičiate pačios spausdintuvo tvarkyklės.
Įsitikinkite, kad spausdintuvą prie savo kompiuterio patikimai prijungėte USB kabeliu, ir įjunkite spausdintuvą.

„Mac“ OS X 10.5 ar vėlesnė versija

Spustelėkite + pridėjimo mygtuką.
Spustelėkite Default, tada pasirinkite spausdintuvą, prie kurio rodoma USB jungtis. Įsitikinkite, kad Print Using meniu rodoma atitinkamo EPSON spausdintuvo tvarkyklė, tada „Printer Browser“ lange spustelėkite Add.
Iš „System Preferences“ meniu pasirinkite Quit System Preferences.

„Mac“ OS X 10.3 ir 10.4 versijos

Spustelėkite Add mygtuką „Print Center“ arba „Printer Setup Utility“ meniu.
Norėdami pasirinkti atitinkamą prievadą ir spausdintuvą, vykdykite nurodymus, susijusius su jūsų naudojama operacine sistema.

„Mac“ OS X 10.3 versija:

Pasirinkite USB kaip prievadą. Tada pasirinkite spausdintuvą ir spustelėkite Add „Printer List“ dialoginiame lange.

„Mac“ OS X 10.4 versija:

Spustelėkite Default Browser, tada pasirinkite spausdintuvą, prie kurio rodoma USB jungtis, ir spustelėkite Add „Printer Browser“ dialoginiame lange.
Pasirinkite Quit Printer Setup Utility iš „Printer Setup Utility“ meniu.
