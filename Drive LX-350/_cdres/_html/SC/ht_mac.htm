按照下列步骤在计算机上添加打印机驱动程序。如果产品带有传真机功能，并且您想从计算机发送传真，请同时添加传真机驱动程序。
对于Mac OS X 10.5或更新版本
对于Mac OS X 10.4.x
注意：
一旦添加了驱动程序，除非您改变了连接方法或使用了其他产品，否则不需要再次进行添加。
确认产品已开启，并已用USB或网络连接将产品牢固地连接至计算机。

对于Mac OS X 10.5或更新版本

使用USB连接时，打印机驱动程序将自动添加并显示在Print & Fax（打印与传真）窗口。仅当打印机驱动程序不显示或通过网络连接打印机时需要添加打印机驱动程序。传真机驱动程序（对于带有传真机功能的机型）不会自动添加。在添加打印机驱动程序的同时添加传真机驱动程序。
单击+添加按钮。
单击Default（预设）（或Default Browser（预设浏览器））。从列表中选择Epson产品，然后单击Add（添加）。查看下表，确认选择了正确的产品。

查找传真机驱动程序可能需要一些时间。等到传真机驱动程序显示在列表中，然后使用相同的步骤添加传真机驱动程序。
从System Preferences（系统预置）菜单中选择Quit System Preferences（退出系统预置）。

对于Mac OS X 10.4.x

若要添加打印机驱动程序，请按照与Mac OS X 10.5或更新版本相同的步骤。
若要添加传真机驱动程序，请按照以下步骤。
单击Default Browser（预设浏览器），然后单击More Printers（更多打印机）按钮。
从Printer Browser（打印机浏览器）下拉菜单中选择EPSON FAX。等到传真机驱动程序（FAX XXXXXX）显示在列表中，然后选择传真机驱动程序并单击Add（添加）。
从System Preferences（系统预置）菜单中选择Quit System Preferences（退出系统预置）。
