Dacă imprimanta nu este recunoscută la instalarea driverului, încercaţi soluţiile date în această secţiune.
1. Imprimanta este pornită?
2. Computerul şi imprimantă sunt conectate corect?

1. Imprimanta este pornită?

Asiguraţi-vă că imprimanta este pornită.
Cablul de alimentare trebuie să fie bine introdus în priza electrică.
Testaţi existenţa tensiunii de reţea prin conectarea unui alt aparat electric la o priză şi pornirea acestuia. Dacă aparatul poate fi pus sub tensiune, imprimanta poate fi defectă.

Dacă imprimanta este oprită:

Porniţi imprimanta, apoi faceţi clic pe butonul Înapoi.

Dacă imprimanta este pornită:

Treceţi la pasul următor.

2. Computerul şi imprimanta sunt conectate corect?

Cablul USB trebuie bine fixat în portul imprimantei şi cel al computerului, şi nu trebuie să prezinte deteriorări sau încreţiri foarte mari.
Imprimanta nu va fi recunoscută la instalarea driverului dacă aceasta este conectată la selector de imprimante sau la o memorie tampon de imprimantă. În acest caz, încercaţi să conectaţi în mod direct imprimanta la computer cu ajutorul cablului USB.
Dacă imprimanta este conectată la computer printr-un hub USB, conectaţi imprimanta la hub-ul first-tier (nivel unu) al computerului. Dacă driverul de imprimantă nu este totuşi recunoscut de computer, încercaţi să conectaţi imprimanta la computer eliminând hub-ul USB.
Dacă imprimanta este conectată la computer printr-un hub USB, asiguraţi-vă că dispozitivul de hub USB este recunoscut de către computer.

Dacă imprimanta şi computerul nu sunt corespunzător conectate, faceţi conexiunea în mod corect, apoi faceţi clic pe butonul Înapoi.
