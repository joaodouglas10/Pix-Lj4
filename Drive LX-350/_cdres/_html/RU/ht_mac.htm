Примечание.
Повторная установка драйверов потребуется только в случае использования другого продукта или изменения метода подключения.
Убедитесь, что продукт включен и правильно подключен к компьютеру с помощью сетевого или USB-соединения.

Для Mac OS X 10.5 или более поздних версий

Если используется соединение USB, драйвер принтера установится автоматически и отобразится в окне Print & Fax (Печать и факс). Ручная установка требуется только в случае, если драйвер принтера не отображается или принтер подключен к компьютеру по сети. Драйвер факса (для моделей с функцией факса) автоматически не устанавливается. Его следует устанавливать так же, как и драйвер принтера.
Нажмите кнопку + (Добавить).
Щелкните Default (По умолчанию) (или Default Browser (Браузер по умолчанию)). Выберите свой продукт Epson из списка и щелкните Add (Добавить). В приведенной ниже таблице проверьте правильность выбора.

* XXXXXX = название продукта, YYYYYY = последние 6 цифр MAC-адреса
Поиск драйвера факса может занять некоторое время. Подождите, пока драйвер факса отобразится в списке, затем установите его, выполнив описанную выше процедуру.
В меню System Preferences (Системные настройки) выберите Quit System Preferences (Закрыть системные настройки).

Для Mac OS X 10.4.x

Чтобы установить драйвер принтера, выполните ту же процедуру, что и для Mac OS X 10.5 или более поздних версий.
Для установки драйвера факса выполните процедуру, приведенную ниже.
Щелкните Default Browser (Браузер по умолчанию), затем нажмите кнопку More Printers (Дополнительные принтеры).
В раскрывающемся меню Printer Browser (Браузер принтеров) выберите EPSON FAX. Подождите, пока драйвер факса (FAX XXXXXX) появится в списке, затем выберите его и щелкните Add (Добавить).
В меню System Preferences (Системные настройки) выберите Quit System Preferences (Закрыть системные настройки).
