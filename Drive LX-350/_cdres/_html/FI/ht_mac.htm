Mac OS X 10.4.x
Huomautus:
Kun olet lisännyt tulostinohjaimet, niitä ei tarvitse lisätä uudelleen, ellet halua vaihtaa yhteysmenetelmää tai tulostinohjainta.
Varmista, että tulostimeen on kytketty virta ja että olet liittänyt tulostimen tietokoneeseen kunnolla USB-kaapelilla tai verkkoyhteydellä.

Mac OS X 10.5 tai uudempi

USB-yhteydessä tulostinohjain lisätään automaattisesti ja näytetään Print & Fax (Tulostus ja faksaus) -ikkunassa. Sinun on lisättävä tulostinohjain vain, jos tulostinohjainta ei näy tai yhdistät tulostimen verkon kautta. Faksiohjainta (faksitoiminnolla varustetuissa malleissa) ei lisätä automaattisesti. Lisää faksiohjain samoin kuin tulostinohjain.
Osoita lisäyspainiketta +.
Osoita Default (Oletus) (tai Default Browser (Oletusselain)). Valitse Epson-tuote luettelosta ja osoita sitten Add (Lisää). Varmista alla olevasta luettelosta tarkistamalla, että olet valinnut oikean tuotteen.

Faksiohjaimen löytäminen voi kestää jonkin aikaa. Odota kunnes faksiohjain näytetään luettelossa, ja lisää sitten faksiohjain samalla tavalla.
Valitse System Preferences (Järjestelmäasetukset) -valikosta Quit System Preferences (Lopeta järjestelmäasetukset).

Mac OS X 10.4.x

Lisää tulostinohjain samalla tavoin kuin Mac OS X 10.5- tai uudemmassa käyttöjärjestelmässä.
Lisää faksiohjain alla olevien ohjeiden mukaisesti.
Osoita Default Browser (Oletusselain) ja osoita sitten More Printers (Muita kirjoittimia) -painiketta.
Valitse Printer Browser (Kirjoitinselain) -vetovalikosta EPSON FAX. Odota kunnes faksiohjain (FAX XXXXXX) näkyy luettelossa. Kun se on näkyvissä, valitse faksiohjain ja osoita sitten Add (Lisää).
Valitse System Preferences (Järjestelmäasetukset) -valikosta Quit System Preferences (Lopeta järjestelmäasetukset).
