Hinweis:
Sobald die Treiber hinzugefügt wurden, müssen diese nicht noch einmal hinzugefügt werden, außer nach dem Ändern der Verbindungsmethode oder bei Verwendung eines anderen Produktes.
Stellen Sie sicher, dass Ihr Gerät eingeschaltet und korrekt mit dem Computer über USB oder einer Netzwerkverbindung verbunden wurde.

Für Mac OS X 10.5 oder später

Bei Verwendung einer USB-Verbindung wird der Druckertreiber automatisch hinzugefügt und im Fenster Print & Fax (Drucken und Faxen) angezeigt. Sie müssen den Druckertreiber nur manuell hinzufügen, wenn entweder der Druckertreiber nicht angezeigt wird oder Sie mit dem Drucker über ein Netzwerk verbunden sind. Der Faxtreiber (nur für Modelle mit Faxfunktion) wird nicht automatisch hinzugefügt. Fügen Sie den Fax- sowie den Druckertreiber hinzu.
Klicken Sie auf die Schaltfläche + (Hinzufügen).
Klicken Sie auf Default (Standard) (oder Default Browser (Standard-Browser)). Wählen Sie Ihr Epson-Produkt aus der Liste und klicken Sie auf Add (Hinzufügen). Überprüfen Sie Ihre Einstellungen mittels der nachfolgenden Tabelle.

Die Suche nach dem Faxtreiber kann eine kurze Zeit beanspruchen. Warten Sie, bis der Faxtreiber in der Liste erscheint und fügen Sie ihn dann mittels derselben Methode hinzu.
Wählen Sie aus dem Menü „System Preferences“ (Systemeinstellungen) die Option Quit System Preferences (Systemeinstellungen beenden).

Für Mac OS X 10.4.x

Um den Druckertreiber hinzuzufügen, folgen Sie denselben Schritten wie unter Mac OS X 10.5 oder später.
Führen Sie die folgenden Schritte aus, um den Druckertreiber hinzuzufügen.
Klicken Sie auf Default Browser (Standard-Browser) und dann auf die Schaltfläche More Printers (Weitere Drucker).
Wählen Sie EPSON FAX aus dem Pulldownmenü Printer Browser (Druckerübersicht). Warten Sie, bis der Faxtreiber (FAX XXXXXX) in der Liste angezeigt wird. Wählen Sie den Faxtreiber, sobald er angezeigt wird, und klicken Sie auf Add (Hinzufügen).
Wählen Sie aus dem Menü „System Preferences“ (Systemeinstellungen) die Option Quit System Preferences (Systemeinstellungen beenden).
