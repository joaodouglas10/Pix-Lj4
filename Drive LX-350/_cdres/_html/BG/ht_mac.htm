Преди печат е необходимо да добавите драйвера за принтер за вашето устройство. Следвайте следните стъпки, за да добавите драйвера за принтер.
За Mac OS X 10.5 или по-нова версия
За Mac OS X 10.3 до 10.4
Забележка:
След като сте добавили драйвера за принтер, не е необходимо да го добавяте отново, освен ако не искате да смените драйвера.
Уверете се, че сте свързали надеждно принтера към компютъра посредством USB кабела, след това включете принтера.

За Mac OS X 10.5 или по-нова версия

Кликнете върху бутона за добавяне +.
Кликнете върху Default и след това изберете принтера, за който е показана USB връзка. Уверете се, че се изобразява правилният драйвер за принтер на EPSON в менюто Print Using и след това кликнете върху Add в прозореца на Printer Browser.
Изберете Quit System Preferences от менюто System Preferences.

За Mac OS X 10.3 до 10.4

Кликнете върху бутона Add в Print Center или Printer Setup Utility.
Следвайте инструкциите за вашата операционна система за избор на съответния порт и принтер.

За Mac OS X 10.3:

Изберете USB като тип порт. След това изберете принтера и кликнете върху Add в диалоговия прозорец Printer List.

За Mac OS X 10.4:

Кликнете върху Default Browser и след това изберете принтера, за който се изобразява USB връзка; накрая кликнете върху Add в диалоговия прозорец Printer Browser.
Изберете Quit Printer Setup Utility от менюто Printer Setup Utility.
