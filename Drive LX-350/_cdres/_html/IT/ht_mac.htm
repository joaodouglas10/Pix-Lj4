Per aggiungere il driver della stampante sul computer, procedere come segue. Se il prodotto è dotato di funzione fax e si desidera inviare fax dal computer, aggiungere anche il driver fax.
Per Mac OS X 10.5 o versione successiva
Per Mac OS X 10.4.x
Nota:
Dopo aver aggiunto i driver, non sarà necessario aggiungerli nuovamente a meno che non venga cambiato il metodo di connessione o si utilizzi un prodotto diverso.
Verificare che il prodotto sia acceso e collegato correttamente al computer con una connessione USB o di rete.

Per Mac OS X 10.5 o versione successiva

Per le connessioni USB, il driver della stampante viene aggiunto automaticamente e visualizzato nella finestra Print & Fax (Stampa e Fax). Il driver della stampante deve essere aggiunto solo se non è visualizzato o se la stampante viene collegata mediante una rete. Il driver fax (per i modelli dotati di funzione fax) non viene aggiunto automaticamente. Aggiungere il driver fax oltre al driver della stampante.
Fare clic sul pulsante di aggiunta +.
Fare clic su Default (o Default Browser (Browser di default)). Selezionare il prodotto Epson dall'elenco e fare clic su Add (Aggiungi). Controllare la seguente tabella per accertarsi di aver selezionato il driver corretto.

La ricerca del driver fax potrebbe richiedere tempo. Attendere che il driver fax venga visualizzato nell'elenco, quindi aggiungere il driver fax utilizzando la stessa procedura.
Selezionare Quit System Preferences (Esci da Preferenze di Sistema) nel menu System Preferences (Preferenze di Sistema).

Per Mac OS X 10.4.x

Per aggiungere il driver della stampante, seguire la stessa procedura indicata per Mac OS X 10.5 o versione successiva.
Per aggiungere il driver fax, procedere come segue.
Fare clic su Default Browser (Browser di default), quindi fare clic sul pulsante More Printers (Altre stampanti).
Dal menu a discesa Printer Browser (Browser stampante), selezionare EPSON FAX. Attendere che il driver fax (FAX XXXXXX) sia visualizzato nell'elenco. Quando viene visualizzato, selezionare il driver fax e fare clic su Add (Aggiungi).
Selezionare Quit System Preferences (Esci da Preferenze di Sistema) nel menu System Preferences (Preferenze di Sistema).
