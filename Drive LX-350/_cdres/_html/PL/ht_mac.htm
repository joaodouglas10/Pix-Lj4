W celu dodania drukarki do komputera posłuż się poniższą procedurą. Jeśli urządzenie obejmuje funkcję faksowania i będzie ona używana, należy również dodać sterownik faksu.
System Mac OS X 10.5 lub nowszy
System Mac OS X 10.4.x
Uwaga:
Po dodaniu sterownika nie ma potrzeby ponownego jego dodawania, o ile nie została zmieniona metoda połączenia lub nie użyto innego urządzenia.
Należy sprawdzić, czy urządzenie jest włączone i odpowiednio połączone z komputerem za pomocą portu USB lub sieci.

System Mac OS X 10.5 lub nowszy

W przypadku połączenia za pomocą portu USB sterownik drukarki zostanie automatycznie dodany i wyświetlony w oknie Print & Fax (Drukarka i faks). Sterownik drukarki należy dodać jedynie w razie, gdy nie jest wyświetlany lub połączenie następuje przez sieć. Sterownik faksu (dla modeli z funkcją faksowania) nie jest dodawany automatycznie. Trzeba go dodać razem ze sterownikiem drukarki.
Kliknij przycisk + (Dodaj).
Kliknij opcję Default (Domyślne) lub Default Browser (Domyślna przeglądarka). Wybierz urządzenie firmy Epson z listy, a następnie kliknij przycisk Add (Dodaj). Sprawdź tabelę poniżej, aby potwierdzić, że została wybrana poprawna pozycja.

Wyszukanie sterownika faksu może zająć pewien czas. Należy odczekać, aż sterownik pojawi się na liście, a następnie dodać go z użyciem tej samej procedury.
Wybierz opcję Quit System Preferences (Zamknij okno preferencji systemowych) z menu System Preferences (Preferencje systemowe).

System Mac OS X 10.4.x

W celu dodania sterownika drukarki należy posłużyć się tą samą procedurą, jaka ma zastosowanie w systemie Mac OS X 10.5 lub nowszym.
Dodanie sterownika faksu umożliwia poniższa procedura.
Kliknij polecenie Default Browser (Domyślna przeglądarka), a następnie kliknij przycisk More Printers (Więcej drukarek).
Z menu rozwijanego Printer Browser (Przeglądarka drukarek) wybierz pozycję EPSON FAX (FAKS EPSON). Odczekaj do momentu pojawienia się sterownika (FAX XXXXXX) na liście. Gdy to nastąpi, wybierz sterownik i kliknij przycisk Add (Dodaj).
Wybierz opcję Quit System Preferences (Zamknij okno preferencji systemowych) z menu System Preferences (Preferencje systemowe).
