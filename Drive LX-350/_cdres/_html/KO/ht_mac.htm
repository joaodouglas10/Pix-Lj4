다음 과정에 따라 컴퓨터에 프린터 드라이버를 추가하십시오. 팩스 기능이 내장된 제품을 이용해 컴퓨터로 팩스를 보내려 할 때는 팩스 드라이브도 추가해야 합니다.
Mac OS X 10.5 이상인 경우
Mac OS X 10.4.x인 경우
참고:
한 번 프린터 드라이버를 추가하면 연결 방법과 다른 제품을 사용하지 않는 이상 다시 추가할 필요는 없습니다.
USB나 네트워크 연결을 사용하여 프린터를 컴퓨터에 적절히 연결하고 프린터를 켰는지 확인합니다.

Mac OS X 10.5 이상인 경우

USB로 연결한 경우, 프린터 드라이버는 자동으로 추가되며 Print & Fax(프린트 & 팩스) 창에 표시됩니다. 프린터 드라이버가 표시되지 않거나 네트워크로 프린터를 연결할 경우 프린터 드라이버를 추가해야 합니다. 팩스 드라이버(팩스 기능 내장 모델)는 자동으로 추가되지 않습니다. 프린터 드라이버와 마찬가지로 팩스 드라이버를 추가합니다.
+ 추가 버튼을 클릭합니다.
Default(기본)(혹은 Default Browser(기본 브라우저))를 클릭합니다. 목록에서 Epson 제품을 선택하고 Add(추가)를 클릭합니다. 다음 표를 참조하여 맞는 상품을 고릅니다.
연결 방법
프린터 드라이버
팩스 드라이버
USB
EPSON XXXXXX
FAX XXXXXX (USB)
네트워크
EPSONYYYYYY (XXXXXX)
FAX EPSONYYYYYY
* XXXXXX = 제품명, YYYYYY = MAC 어드레스 끝 6자리
팩스 드라이버를 검색하는데 시간이 걸릴 수 있습니다. 목록에 팩스 드라이버가 표시될 때까지 기다린 다음 같은 과정으로 팩스 드라이버를 추가합니다.
System Preferences(시스템 환경설정) 메뉴에서 Quit System Preferences(시스템 환경설정 종료)를 선택합니다.

Mac OS X 10.4.x인 경우

컴퓨터 사양이 Mac OS X 10.5 이상일 때 같은 과정으로 프린터 드라이버를 추가합니다.
다음 과정에 따라 팩스 드라이버를 추가합니다.
Default Browser(기본 브라우저)를 클릭한 다음 More Printers(다른 프린터) 버튼을 클릭합니다.
Printer Browser(프린터 브라우저)에서 메뉴를 내려 EPSON FAX를 선택합니다. 목록에 팩스 드라이버(FAX XXXXXX(팩스 XXXXXX))가 표시될 때까지 기다립니다. 표시되면 팩스 드라이버를 선택하고 Add(추가)를 클릭합니다.
System Preferences(시스템 환경설정) 메뉴에서 Quit System Preferences(시스템 환경설정 종료)를 선택합니다.
