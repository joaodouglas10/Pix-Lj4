Merk:
Når du har lagt til driveren(e), behøver du ikke å legge den til igjen, bortsett fra hvis du endrer tilkoblingsmetode eller bruker et annet produkt.
Kontroller at produktet er slått på og koblet skikkelig til datamaskinen med USB eller en nettverkstilkobling.

For Mac OS X 10.5 eller nyere

Skriverdriveren legges automatisk til for USB-tilkoblinger og vises i Print & Fax (Utskrift og faks)-vinduet. Du trenger bare å legge til skriverdriveren hvis den ikke vises, eller hvis du kobler til skriveren via et nettverk. Faksdriveren (for modeller med faksfunksjon) legges ikke til automatisk. Legg til faksdriveren i tillegg til skriverdriveren.
Klikk Legg til +-knappen.
Klikk Default (Standard) (eller Default Browser (Standardnavigering)). Velg Epson-produktet fra listen, og klikk deretter Add (Legg til). Se i tabellen under om du har valgt riktig.

Det kan ta litt tid å søke etter faksdriveren. Vent til faksdriveren vises i listen, og legg deretter til faksdriveren ved hjelp av samme fremgangsmåte.
Velg Quit System Preferences (Avslutt Systemvalg) fra System Preferences (Systemvalg)- menyen.

For Mac OS X 10.4.x

Følg same fremgangsmåte som for Mac OS X 10.5 eller nyere for å legge til skriverdriveren.
Følg fremgangsmåten under for å legge til faksdriveren.
Klikk Default Browser (Standardnavigering) og klikk deretter More Printers (Flere skrivere)-knappen.
Velg EPSON FAX fra Printer Browser (Skrivernavigeringsvindu)-nedtrekksmenyen. Vent til faksdriveren (FAX XXXXXX) vises i listen. Velg faksdriveren når den vises, og klikk deretter Add (Legg til).
Velg Quit System Preferences (Avslutt Systemvalg) fra System Preferences (Systemvalg)- menyen.
