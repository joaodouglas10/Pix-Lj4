Nota:
Una vez que haya agregado el/los controlador/es de impresora, no tiene que volver a agregarlo/s a menos que cambio el método de conexión o utilice un producto diferente.
Compruebe que la impresora está encendida y conectada correctamente al ordenador mediante una conexión USB o de red.

Para Mac OS X 10.5 o posterior

Para conexiones USB, el controlador de la impresora se agrega automáticamente y se visualiza en la ventana Print & Fax (Impresión y Fax). Sólo necesita agregar el controlador de la impresora si éste no se visualiza o si ha conectado la impresora en red. El controlador de fax (para modelos con función de fax) no se agrega automáticamente. Agregue el controlador del fax y el de la impresora.
Haga clic en el botón + (agregar).
Haga clic en Default (Por omisión) (o Default Browser (Navegador por omisión)). Seleccione su producto Epson de la lista y haga clic en Add (Añadir). Compruebe la tabla que se muestra más abajo para asegurarse de que ha seleccionado el producto correcto.

La búsqueda del controlador de fax puede tardar unos instantes. Espere hasta el se visualice el controlador de fax en la lista y añada el controlador siguiendo el mismo proceso.
Seleccione Quit System Preferences (Salir de Preferencias del Sistema) desde el menú System Preferences (Preferencias del Sistema).

Para Mac OS X 10.4.x

Para agregar el controlador de la impresora, siga el mismo proceso que para Mac OS X 10.5 o posterior.
Para agregar el controlador de fax, siga el siguiente proceso.
Haga clic en Default Browser (Navegador por omisión) y luego haga clic en el botón More Printers (Más impresoras).
El menú desplegable de Printer Browser (Navegador de impresoras), selecciona EPSON FAX. Espere hasta que se visualice en la lista el controlador de fax (FAX XXXXXX). Cuando se visualice, seleccione el controlador de fax y haga clic en Add (Añadir).
Seleccione Quit System Preferences (Salir de Preferencias del Sistema) desde el menú System Preferences (Preferencias del Sistema).
