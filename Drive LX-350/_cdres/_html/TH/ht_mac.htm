จำเป็นต้องติดตั้งไดร์เวอร์สำหรับพรินเตอร์ลงในเครื่องก่อนที่จะเริ่มพิมพ์ได้ ทำตามขั้นตอนด้านล่างเพื่อติดตั้งไดร์เวอร์สำหรับพรินเตอร์
สำหรับ Mac OS X 10.5 หรือใหม่กว่า
สำหรับ Mac OS X 10.3 ถึง 10.4
หมายเหตุ:
เมื่อติดตั้งไดร์เวอร์สำหรับพรินเตอร์แล้ว คุณไม่ต้องติดตั้งซ้ำอีก เว้นแต่จะมีการเปลี่ยนไดร์เวอร์สำหรับพรินเตอร์
ตรวจดูให้แน่ใจว่าได้เชื่อมต่อพรินเตอร์กับคอมพิวเตอร์ด้วยสาย USB แน่นดีแล้ว จากนั้นจึงเปิดคอมพิวเตอร์

สำหรับ Mac OS X 10.5 หรือใหม่กว่า

คลิกปุ่มเพิ่ม +
คลิก Default แล้วเลือกพรินเตอร์ที่มีการเชื่อมต่อแบบ USB ตรวจดูว่ามีไดร์เวอร์สำหรับพรินเตอร์ EPSON ที่ถูกต้องปรากฏในเมนู Print Using แล้วคลิก Add ในหน้าต่าง Printer Browser
เลือก Quit System Preferences จากเมนู System Preferences

สำหรับ Mac OS X 10.3 ถึง 10.4

คลิกปุ่ม Add ใน Print Center หรือ Printer Setup Utility
ปฏิบัติตามคำแนะนำของระบบปฏิบัติการเพื่อเลือกพอร์ตและพรินเตอร์ที่ถูกต้อง

สำหรับ Mac OS X 10.3:

เลือกพอร์ต USB แล้วเลือกพรินเตอร์ และคลิก Add ในหน้าต่าง Printer List

สำหรับ Mac OS X 10.4:

คลิก Default Browser แล้วเลือกพรินเตอร์ที่มีการเชื่อมต่อแบบ USB จากนั้นคลิก Add ในหน้าต่าง Printer Browser
เลือก Quit Printer Setup Utility จากเมนู Printer Setup Utility
