لازم است قبل از استفاده از دستگاه چاپگر، درایور آنرا اضافه نمایید. مراحل ذیل را برای اضافه کردن درایور چاپگر انجام دهید.
برای نسخه 10.5 یا بالاتر Mac OS X
برای نسخه 10.3 تا 10.4 Mac OS X
توجه:
پس از اضافه کردن درایور چاپگر، احتیاجی به اضافه کردن مجدد آن نیست، مگر اینکه درایورهای چاپگر را تغییر دهید.
پس از اطمینان از اتصال صحیح چاپگر به رایانه تان از طریق کابل USB، آنرا روشن کنید.

برای نسخه 10.5 یا بالاتر Mac OS X

روی گزینه + اضافه کردن کلیک کنید.
روی دکمه Default کلیک کرده و سپس چاپگری را که اتصال USB برای آن نشان داده شده است، انتخاب کنید. مطمئن شوید درایور چاپگر اپسون مناسب در منوی Print Using نمایش داده شده و سپس بر روی Add در پنجره Printer Browser کلیک کنید.
Quit System Preferences را از منوی System Preferences انتخاب کنید.

برای نسخه 10.3 تا 10.4 Mac OS X

بر روی دکمه Add در Print Center یا Printer Setup Utility کلیک کنید.
برای انتخاب پورت یا چاپگر مناسب، دستورات مربوط به سیستم عامل خود را دنبال کنید.

برای نسخه 10.3 Mac OS X:

پورت USB را انتخاب کنید. سپس چاپگر را انتخاب نموده و بر روی دکمه Add در پنجره Printer List کلیک کنید.

برای نسخه 10.4 Mac OS X:

بر روی Default Browser کلیک کرده، سپس چاپگری را که اتصال USB برای آن نشان داده شده است، انتخاب کنید، پس از آن بر روی Add در پنجره Printer Browser کلیک کنید.
گزینه Quit Printer Setup Utility را از منوی Printer Setup Utility انتخاب کنید.
