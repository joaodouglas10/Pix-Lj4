Потребно е да го додадете драјверот на печатачот за вашиот производ пред да печатите. Следете ги чекорите долу за да го додадете драјверот на печатачот.
За Mac OS X 10.5 или понов
За Mac OS X 10.3 до 10.4
Белешка:
Кога еднаш ќе го додадете драјверот на печатачот, нема потреба од повторно да го додавате освен ако не смените драјвери на печатачот.
Проверете дали е добро поврзан печатачот со компјутерот преку USB кабелот, а потоа вклучете го печатачот.

За Mac OS X 10.5 или понов

Кликнете на копчето за додавање +.
Кликнете на Default, а потоа одберете го печатачот за кој се појавува USB врската. Проверете дали се појавува соодветен EPSON драјвер на печатач во менито Print Using, а потоа кликнете на Add во прозорецот Printer Browser.
Од менито System Preferences одберете Quit System Preferences.

За Mac OS X 10.3 до 10.4

Кликнете на копчето Add во Print Center или Printer Setup Utility.
Следете ги упатствата за вашиот оперативен систем за да го одберете соодветниот порт и печатач.

За Mac OS X 10.3:

Одберете USB како избор за порт. Потоа одберете го печатачот и кликнете на Add во дијалогот Printer List.

За Mac OS X 10,4:

Кликнете на Default Browser, потоа одберете го печатачот за кој се појавува врската USB и кликнете на Add во дијалогот Printer Browser.
Од менито Printer Setup Utility одберете Quit Printer Setup Utility.
