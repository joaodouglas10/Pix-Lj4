Pre štampanja potrebno je da dodate upravljački program štampača. Sledite ove korake da biste dodali upravljački program štampača.
Za Mac OS X 10.5 ili noviju verziju
Za Mac OS X 10.3 do 10.4
Napomena:
Nakon što dodate upravljački program štampača, više neće biti potrebe da ga ponovo dodajete, sem ako ne promenite upravljačke programe štampača.
Uverite se da je štampač bezbedno priključen na računar pomoću USB kabla i zatim uključite štampač.

Za Mac OS X 10.5 ili noviju verziju

Kliknite na dugme + dodaj.
Kliknite na Default i zatim izaberite štampač za koji je prikazana USB veza. Uverite se da je u meniju Print Using prikazan odgovarajući upravljački program EPSON štampača i zatim u prozoru Printer Browser kliknite na Add.
U meniju System Preferences izaberite Quit System Preferences.

Za Mac OS X 10.3 do 10.4

U Print Center ili Printer Setup Utility kliknite na dugme Add.
Sledite uputstva za operativni sistem da biste odabrali odgovarajući port i štampač.

Za Mac OS X 10.3:

Prilikom biranja porta izaberite USB. Zatim izaberite štampač i kliknite na Add u dijalogu Printer List.

Za Mac OS X 10.4:

Kliknite na Default Browser i zatim izaberite štampač za koji je prikazana USB veza, a potom u dijalogu Printer Browser kliknite na Add.
U meniju Printer Setup Utility izaberite Quit Printer Setup Utility.
