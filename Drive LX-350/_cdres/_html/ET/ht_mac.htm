Enne printimist on vajalik lisada tootele printeri draiver. Järgige neid samme, et lisada printeri draiverit.
Mac OS X 10.5 või uuema puhul
Mac OS X 10.3 kuni 10.4 puhul
Märkus.
Pärast printeri draiveri lisamist ei pea te seda uuesti tegema, välja arvatud juhul, kui vahetate printeri draivereid.
Veenduge, et olete printeri ja arvuti õigesti USB-kaabli abil ühendanud ning lülitage seejärel printer sisse.

Mac OS X 10.5 või uuema puhul

Klõpsake nuppu + (lisa).
Klõpsake Default ning seejärel valige printer, mille USB-ühendus on kuvatud. Veenduge, et Print Using menüüs on kuvatud vastav EPSONI printer ning seejärel klõpsake aknas Printer Browser nuppu Add.
Valige menüüst System Preferences valik Quit System Preferences.

Mac OS X 10.3 kuni 10.4 puhul

Klõpsake Print Center'is või Print Setup Utility'is nuppu Add.
Järgige oma operatsioonisüsteemi kohta käivaid juhiseid, et valida sobiv port ja printer.

Mac OS X 10.3 puhul:

Valige pordiks USB. Seejärel valige dialoogiboksis Printer List printer ja klõpsake nuppu Add.

Mac OS X 10.4 puhul:

Klõpsake Default Browser ning seejärel valige printer, mille USB-ühendus on kuvatud ja siis klõpsake dialoogiboksis Printer Browser nuppu Add.
Valige menüüst Printer Setup Utility valik Quit Printer Setup Utility.
