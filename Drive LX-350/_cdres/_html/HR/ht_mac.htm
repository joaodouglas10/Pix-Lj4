Prije ispisivanja potrebno je dodati upravljački program pisača. Za dodavanje upravljačkog programa pisača slijedite ove korake.
Za Mac OS X 10.5 ili noviju inačicu
Za Mac OS X 10.3 do 10.4
Napomena:
Nakon dodavanja upravljačkog programa pisača, neće biti potrebno ponovno ga dodavati, osim ako ne promijenite upravljačke programe pisača.
Uvjerite se da je pisač sigurno spojen s računalom putem USB kabela, a zatim uključite pisač.

Za Mac OS X 10.5 ili noviju inačicu

Kliknite na gumb + add.
Kliknite na Default te odaberite pisač za koji je prikazana USB veza. Uvjerite se da je u izborniku Print Using prikazan odgovarajući upravljački program EPSON pisača te kliknite na Add u prozoru Printer Browser.
U izborniku System Preferences odaberite Quit System Preferences.

Za Mac OS X 10.3 do 10.4

U Print Center ili Printer Setup Utility kliknite na gumb Add.
Slijedite upute za operativni sustav da biste odabrali odgovarajući priključak i pisač.

Za Mac OS X 10.3:

Pri odabiru priključka odaberite USB. Potom odaberite pisač i kliknite na Add u dijalogu Printer List.

Za Mac OS X 10.4:

Kliknite na Default Browser te odaberite pisač za koji je prikazana USB veza, a potom u dijalogu Printer Browser kliknite na Add.
U izborniku Printer Setup Utility odaberite Quit Printer Setup Utility.
