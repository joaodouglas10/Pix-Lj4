Mac OS X v10.5.x〜v10.6.xの場合

USB接続では、プリンタードライバーは自動的に追加され[プリントとファクス]画面に表示されます。ネットワーク接続、および、ファクスドライバー（ファクス対応機種のみ）は自動では追加されません。以下の手順に従ってプリンタードライバー（IP)とファクスドライバー（FAX)を追加してください。
［＋］ボタンをクリックします。
［デフォルト］をクリックし、追加するプリンターを選択して［追加］をクリックします。以下の表示ルールを参照し、適切なものを選択してください。

* XXXXXX = 製品名、YYYYYY = MACアドレス下6桁、 ZZZ.ZZZ.ZZZ.ZZZ = IPアドレス
ファクスドライバーが表示されるまでには時間がかかることがあります。リストにファクスドライバーが表示された後、プリンタードライバーと同様の手順でファクスドライバーも追加します。
プリンターの一覧に追加されたら、［プリントとファクス］画面を閉じます。

Mac OS X v10.4.xの場合

プリンタードライバーの追加方法は、Mac OS X 10.5.xと同様です。 ファクスドライバーを追加するには、以下の手順に従ってください。
［デフォルトブラウザ］をクリックし、［ほかのプリンタ...］ボタンをクリックします。
［プリンタブラウザ］ウィンドウのプルダウンメニューから「EPSON FAX」を選択し、表示されたファクスドライバーを選択して、［追加］をクリックします。
