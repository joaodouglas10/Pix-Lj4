För Mac OS X 10.4.x
Anmärkning:
När skrivardrivrutinen/skrivardrivrutinerna har lagts till en gång behöver den/de inte läggas till igen, såvida du inte ändrar anslutningsmetod eller använder en annan produkt.
Kontrollera att produkten har slagits på och har anslutits på rätt sätt till datorn med hjälp av en USB- eller nätverkskabel.

För Mac OS X 10.5 eller senare

För USB-anslutningar: Skrivardrivrutinen läggs till automatiskt och visas i fönstret Print & Fax (Skrivare och fax). Du behöver bara lägga till skrivardrivrutinen om skrivardrivrutinen inte visas eller om du ansluter skrivaren via ett nätverk. Faxdrivrutinen (för modeller med faxfunktion) läggs inte till automatiskt. Lägg till både fax- och skrivardrivrutinen.
Klicka på knappen Lägg till +.
Klicka på Default (Förval) (eller Default Browser (Förvald bläddrare)). Markera Epson-produkten i listan och klicka sedan på Add (Lägg till). Kontrollera tabellen nedan för att försäkra dig om att du har markerat rätt alternativ.

Det kan ta en stund att leta efter faxdrivrutinen. Vänta tills faxdrivrutinen visas i listan och lägg sedan till faxdrivrutinen genom att tillämpa samma förfaringssätt.
Välj Quit System Preferences (Avsluta systeminställningar) från menyn System Preferences (Systeminställningar).

För Mac OS X 10.4.x

Tillämpa samma förfaringssätt som för Mac OS X 10.5 eller senare för att lägga till skrivardrivrutinen.
Följ instruktionerna nedan för att lägga till faxdrivrutinen.
Klicka på Default Browser (Förvald bläddrare) och klicka sedan på knappen More Printers (Fler skrivare).
Välj EPSON FAX från rullgardinsmenyn Printer Browser (Utskrifter). Vänta tills faxdrivrutinen (FAX XXXXXX) visas i listan. När den visas markerar du faxdrivrutinen och klickar sedan på Add (Lägg till).
Välj Quit System Preferences (Avsluta systeminställningar) från menyn System Preferences (Systeminställningar).
