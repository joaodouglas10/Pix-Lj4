Pred tlačou musíte pridať ovládač tlačiarne pre váš model. Ovládač tlačiarne môžete pridať nasledujúcim postupom.
Pre systém Mac OS X 10.5 alebo novší
Pre systémy Mac OS X 10.3 až 10.4
Poznámka:
Ak ste zadali ovládač tlačiarne, nemusíte ho znova zadať, iba ak chcete zmeniť ovládače tlačiarní.
Uistite sa, že tlačiareň je bezpečne pripojený k počítaču pomocou kábla USB, a zapnite tlačiareň.

Pre systém Mac OS X 10.5 alebo novší

Kliknite na tlačidlo +.
Kliknite na tlačidlo Default a vyberte tlačiareň, pre ktorú je zobrazené pripojenie USB. Uistite sa, že v ponuke Print Using je zobrazený príslušný ovládač tlačiarne EPSON, a kliknite na tlačidlo Add v okne Printer Browser.
V ponuke System Preferences zvoľte Quit System Preferences.

Pre systémy Mac OS X 10.3 až 10.4

V okne Print Center alebo Printer Setup Utility kliknite na tlačidlo Add.
Postupujte podľa pokynov pre váš operačný systém a vyberte príslušný port a tlačiareň.

Pre systém Mac OS X 10.3:

Pri výbere portu zvoľte USB. Vyberte tlačiareň a v dialógovom okne Printer List kliknite na tlačidlo Add.

Pre systém Mac OS X 10.4:

Kliknite na tlačidlo Default Browser, vyberte tlačiareň, pre ktorú je zobrazené pripojenie USB, a v dialógovom okne Printer Browser kliknite na tlačidlo Add.
V ponuke Printer Setup Utility zvoľte Quit Printer Setup Utility.
