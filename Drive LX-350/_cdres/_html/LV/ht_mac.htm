Pirms drukāšanas izstrādājumam ir jāpievieno printera dzinis. Lai pievienotu printera dzini, izpildiet turpmāk aprakstītās darbības.
Operētājsistēmas versijai Mac OS X 10.5 vai jaunākai
Operētājsistēmas versijai no Mac OS X 10.3 līdz 10.4
Piezīme.
Kad pievienosiet printera dzini, tas būs jādara vēlreiz tikai tad, ja mainīsiet printeri.
Pārliecinieties, ka, izmantojot USB kabeli, printeris ir droši savienots ar datoru.

Operētājsistēmas versijai Mac OS X 10.5 vai jaunākai

Klikšķiniet uz + pievienošanas pogas.
Klikšķiniet uz pogas Default un pēc tam atlasiet printeri, kuram ekrānā parādīts USB savienojums. Pārliecinieties, ka izvēlnē Print Using ir parādīts pareizais EPSON printera dzinis un pēc tam logā "Printer Browser" klikšķiniet uz Add.
Izvēlnē "System Preferences" atlasiet Quit System Preferences.

Operētājsistēmas versijai no Mac OS X 10.3 līdz 10.4

Izvēlnē "Print Center" vai "Printer Setup Utility" klikšķiniet uz pogas Add.
Lai atlasītu pareizo portu un printeri, izpildiet atbilstošajai operētājsistēmai sniegtos norādījumus.

Operētājsistēmai Mac OS X 10.3:

Kā portu izvēlieties USB. Pēc tam atlasiet printeri un dialogā "Printer List" klikšķiniet uz Add.

Operētājsistēmai Mac OS X 10,4:

Klikšķiniet uz Default Browser, pēc tam atlasiet printeri, kuram parādīts USB savienojums un tad dialogā "Printer Browser" klikšķiniet uz Add.
Izvēlnē "Printer Setup Utility" atlasiet Quit Printer Setup Utility.
