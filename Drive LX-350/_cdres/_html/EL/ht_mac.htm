Είναι απαραίτητη η προσθήκη του προγράμματος οδήγησης του συγκεκριμένου εκτυπωτή για την εκτύπωση. Ακολουθήστε τα παρακάτω βήματα για να προσθέσετε το πρόγραμμα οδήγησης του εκτυπωτή.
Για Mac OS X 10.5 ή νεότερη έκδοση
Για Mac OS X 10.3 έως 10.4
Σημείωση:
Μετά την προσθήκη του προγράμματος οδήγησης του εκτυπωτή, δεν χρειάζεται να το προσθέσετε ξανά εκτός εάν αλλάξετε πρόγραμμα οδήγησης.
Βεβαιωθείτε ότι ο εκτυπωτής είναι σωστά συνδεδεμένος με τον υπολογιστή σας μέσω του καλωδίου USB και κατόπιν ενεργοποιήστε τον εκτυπωτή.

Για Mac OS X 10.5 ή νεότερη έκδοση

Κάντε κλικ στο κουμπί + add.
Πατήστε Default και στη συνέχεια επιλέξτε τον εκτυπωτή που αντιστοιχεί στη σύνδεση USB που εμφανίζεται. Βεβαιωθείτε ότι εμφανίζεται το σωστό πρόγραμμα οδήγησης του εκτυπωτή EPSON στο μενού Print Using και κατόπιν πατήστε Add στο παράθυρο προσθήκης εκτυπωτή.
Επιλέξτε Quit System Preferences από το μενού System Preferences.

Για Mac OS X 10.3 έως 10.4

Πατήστε το κουμπί Add στο παράθυρο Print Center ή Printer Setup Utility.
Ακολουθήστε τις οδηγίες για το λειτουργικό σας σύστημα και επιλέξτε την κατάλληλη θύρα και τον κατάλληλο εκτυπωτή.

Για Mac OS X 10.3:

Επιλέξτε USB για τη θύρα. Στη συνέχεια επιλέξτε τον εκτυπωτή και πατήστε Add στο πλαίσιο διαλόγου Printer List.

Για Mac OS X 10.4:

Πατήστε Default Browser και στη συνέχεια επιλέξτε τον εκτυπωτή που αντιστοιχεί στη σύνδεση USB που εμφανίζεται. Κατόπιν πατήστε Add στο πλαίσιο διαλόγου Printer Browser.
Επιλέξτε Quit Printer Setup Utility από το μενού Printer Setup Utility.
