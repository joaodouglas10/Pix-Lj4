Перед початком друкування на цьому принтері необхідно додати для нього драйвер. Щоб додати драйвер принтера, виконайте описані нижче дії.
У Mac OS X 10.5 та пізніших версій
У Mac OS X 10.3 і 10.4
Примітка.
Драйвер принтера додається лише один раз. Повторне виконання цієї процедури необхідно лише в разі заміни драйверів принтера.
Перевірте, чи надійно підключений принтер до комп'ютера за допомогою кабелю USB, а потім вмикайте принтер.

У Mac OS X 10.5 та пізніших версій

Натисніть кнопку + (додати).
Натисніть кнопку Default і виберіть принтер, для якого показано підключення через USB. У вікні Printer Browser перевірте, чи правильно вибраний у меню Print Using принтер EPSON, а потім натисніть кнопку Add.
Виберіть у меню System Preferences пункт Quit System Preferences.

У Mac OS X 10.3 і 10.4

Натисніть кнопку Add у вікні Print Center або у вікні Printer Setup Utility.
Виберіть відповідний порт і принтер згідно з інструкціями для операційної системи, яка використовується.

У ОС Mac OS X 10.3:

Виберіть порт USB. У діалоговому вікні Printer List виберіть принтер і натисніть кнопку Add.

У ОС Mac OS X 10.4:

У діалоговому вікні Printer Browser натисніть кнопку Default Browser і виберіть принтер, для якого показано підключення через USB, а тоді натисніть кнопку Add.
Виберіть у меню Printer Setup Utility пункт Quit Printer Setup Utility.
