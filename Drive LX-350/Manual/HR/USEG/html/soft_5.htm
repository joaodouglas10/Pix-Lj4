Korištenje programa EPSON Status Monitor 3 (nadzor stanja)

Postavljanje programa EPSON Status Monitor 3 (nadzor stanja)
Pristupanje programu EPSON Status Monitor 3 (nadzor stanja)
EPSON Status Monitor 3 (nadzor stanja) omogućava nadzor stanja pisača, upozorava na pogreške pisača i nudi upute za rješavanje problema, kada su one potrebne.
EPSON Status Monitor 3 (nadzor stanja) je dostupan kada je:
Pisač izravno priključen na glavno računalo preko paralelnog priključka [LPT1] ili USB priključka.
Sustav podešen tako da podržava dvosmjernu komunikaciju.
EPSON Status Monitor 3 (nadzor stanja) je instaliran kada je pisač izravno spojen, a instalirali ste upravljački program pisača prema uputama u odjeljku Start Here ili Instalacijskom priručniku. Ako pisač zajednički koristite, namjestite EPSON Status Monitor 3 (nadzor stanja) tako da se zajednički pisač može nadzirati na poslužitelju i na klijentima pisača. Pogledajte dio Postavljanje programa EPSON Status Monitor 3 (nadzor stanja) i Korištenje pisača preko mreže.
Oprez:
Iako s udaljene lokacije možete ispisivati na pisač koji je izravno spojen na računalo koristeći funkciju Remote Desktop (Udaljena radna površina)* sustava Windows 7, Vista ili XP, može se pojaviti pogreška u komunikaciji.
* Funkcija Remote Desktop (Udaljena radna površina): Funkcija koja korisniku omogućava pristup aplikacijama ili datotekama na računalu spojenom na uredsku mrežu s pokretnog računala na udaljenoj lokaciji.
Napomena:
Ako koristite računalo s operativnim sustavima Windows 7, Vista ili XP s više prijavljenih korisnika koji istodobno nadziru pisače može se pojaviti poruka o pogreški komunikacije.
Ako je EPSON Status Monitor 3 (nadzor stanja) instaliran s uključenom funkcijom Windows vatrozida (preporučuje se), zajednički pisač možda neće biti moguće nadzirati. Taj problem može se riješiti dodavanjem EEBAgent.exe u Exceptions (Iznimke) funkcije vatrozida.

Postavljanje programa EPSON Status Monitor 3 (nadzor stanja)

Postupak za postavljanje programa EPSON Status Monitor 3 (nadzor stanja):
Za Windows 7:
Kliknite na Start (Početak), a zatim na Devices and Printers (Uređaji i pisači).
Za Windows Vista:
Kliknite na Start (Početak), zatim na Control Panel (Upravljačka ploča) te Hardware and Sound (Hardver i zvuk), a potom na Printers (Pisači).
Za izdanje Windows XP Professional:
Kliknite na Start (Početak) i zatim na Printers and Faxes (Pisači i faksovi).
Za izdanje Windows XP Home:
Kliknite na Start (Početak), zatim na Control Panel (Upravljačka ploča) te na Printers and Faxes (Pisači i faksovi).
Za Windows 2000:
Kliknite na Start (Početak), idite na Settings (Postavke) i zatim kliknite na Printers (Pisači).
Desnom tipkom kliknite ikonu pisača, kliknite Printer properties (Svojstva pisača) (Windows 7) ili Properties (Svojstva) (Windows Vista, XP i 2000), a zatim kliknite karticu Utility.
Pritisnite gumb Monitoring Preferences. Prikazat će se dijaloški okvir Monitoring Preferences.
Dostupne su sljedeće postavke:
Select Notification (Odaberite obvijest)
Prikazuje stanje On/Off potvrdnog okvira greške.
Uključite ga želite li prikazati odabranu obavijest o pogreški.
Select Shortcut Icon (Odaberite ikonu kratice)
Odabrana ikona prikazuje se na programskoj traci. Primjer postavke prikazan je u desnom prozoru. Pritiskom ikone prečaca brže možete otvoriti dijaloški okvir Monitoring Preferences.
Receive error notification for shared printers (Primi obavijest o pogreški za zajedničke pisače)
Uključivanjem ovog potvrdnog okvira mogu se primati obavijesti o pogreškama mrežnih pisača.
Allow monitoring of shared printer (Omogući nadzor zajedničkog pisača)
Uključivanjem ovog potvrdnog okvira ostala računala mogu nadzirati stanje mrežnog pisača.
Napomena:
Sustavu Windows morate pristupiti kao administrator.
Napomena:
Pritiskom gumba Default sve će stavke biti vraćene na zadane postavke.

Pristupanje programu EPSON Status Monitor 3 (nadzor stanja)

Kako biste pristupili programu EPSON Status Monitor 3 (nadzor stanja) učinite nešto od sljedećeg:
Dvaput pritisnite ikonu pisača na programskoj traci. Tu ikonu možete dodati tako da otvorite izbornik Utility i slijedite upute.
Otvorite izbornik Utility i pritisnite ikonu EPSON Status Monitor 3. Informacije o tome kako otvoriti izbornik Pomoćni program pogledajte u Korištenje upravljačkog programa pisača.
Kada pristupite programu EPSON Status Monitor 3 (nadzor stanja) prema gornjim uputama, prikazat će se sljedeći prozor sa stanjem pisača.
U njemu možete pogledati informacije o stanju pisača.
Napomena:
Stanje pisača možda neće biti moguće dohvatiti tijekom ispisivanja. U tom slučaju kliknite gumb EPSON Status Monitor 3 (nadzor stanja) na kartici Utility i koristite pisač s otvorenim prozorom stanja.
