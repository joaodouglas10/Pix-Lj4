User’s Guide Help

Finding Your Way Around

This guide includes links to make moving between sections easier. Links appear as colored, underlined text. When you place your cursor over a link, it changes to a hand icon. Click the link to access a related topic or detailed information.
You can also click these links to move around the guide:
Home
Returns to the opening page.
Contents
Opens the Table of Contents.
You can close the window by clicking the  Close button.
Index
Opens the Index.
You can close the window by clicking the  Close button.
Search
Opens the Search window, where you can locate topics by entering keywords.
Top
Returns to the top of the current page.

Printing a Topic

To print the page you are currently on, open the File menu and click Print in your browser.

Solving Display Problems

If a page in the guide displays incorrectly, click the Refresh or Reload button in your browser.
