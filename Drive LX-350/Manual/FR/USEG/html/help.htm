Guide d'utilisation de l'aide

Comment vous déplacer dans ce guide ?

Le présent guide comprend des liens qui facilitent la navigation entre les sections. Les liens apparaissent sous forme de texte en couleur et souligné. Lorsque vous placez le curseur de la souris sur un lien, il prend la forme d’une main. Cliquez sur le lien pour accéder à la rubrique connexe ou à des informations détaillées.
Vous pouvez également cliquer sur les liens suivants pour vous déplacer dans le guide :
Accueil
Permet de retourner à la page d'accueil.
Table des matières
Permet d'ouvrir la table des matières.
Vous pouvez fermer la fenêtre en cliquant sur le bouton Fermer .
Index
Permet d'ouvrir l'index.
Vous pouvez fermer la fenêtre en cliquant sur le bouton Fermer .
Rechercher
Permet d'ouvrir la fenêtre de recherche dans laquelle vous pouvez localiser des rubriques en saisissant des mots-clés.
Haut
Permet de retourner au début de la page actuellement affichée.

Impression d’une rubrique

Pour imprimer la page actuellement affichée, ouvrez le menu File (Fichier) et cliquez sur Print (Imprimer) dans le navigateur.

Résolution des problèmes d’affichage

Si une page du guide n'est pas affichée correctement, cliquez sur le bouton Refresh (Rafraîchir) ou Reload (Actualiser) du navigateur.
