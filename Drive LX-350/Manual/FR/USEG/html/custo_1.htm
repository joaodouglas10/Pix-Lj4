Site Internet de support technique

Le site Web de support technique d’Epson offre des solutions aux problèmes qui ne peuvent pas être résolus en consultant la section de dépannage de la documentation du produit. Si vous disposez d'un navigateur Web et si vous pouvez vous connecter à Internet, accédez au site :
http://support.epson.net/
Pour obtenir les derniers pilotes, consulter les questions fréquemment posées, visualiser les manuels et télécharger des fichiers, visitez le site :
http://www.epson.com
Sélectionnez ensuite la section d'assistance de votre site Web Epson local.
