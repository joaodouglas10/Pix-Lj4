Despre software-ul de imprimantă

Software-ul Epson include software-ul driverului de imprimantă şi EPSON Status Monitor 3.
Driverul de imprimantă este un software care permite computerului să controleze imprimanta. Trebuie să instalaţi driverul de imprimantă pentru ca aplicaţiile dumneavoastră Windows să beneficieze complet de capacităţile imprimantei.
Programul EPSON Status Monitor 3 vă permite să monitorizaţi starea imprimantei dumneavoastră, vă avertizează când apare o eroare şi vă oferă sfaturi de depanare. EPSON Status Monitor 3 este instalat automat atunci când instalaţi driverul de imprimantă.
Notă:
Pentru a continua, asiguraţi-vă că aveţi deja instalat driverul de imprimantă pe computerul dvs., descris în Start Here sau Ghidul de configurare.
