Verwendung von Bi-D Adjustment (Bidirektionale Einstellung)

Anpassung mit Bi-D Adjustment (Bidirektionale Einstellung)
Dieses Werkzeug ermöglicht Ihnen das Korrigieren der Ausrichtung von vertikalen Linien.

Anpassung mit Bi-D Adjustment (Bidirektionale Einstellung)

Befolgen Sie zur Anpassung von Bi-D Adjustment (Bidirektionale Einstellung) die nachstehenden Schritte.
Vorsicht:
Schalten Sie den Drucker nicht aus, während er ein Ausrichtungsmuster druckt. Andernfalls könnte der Drucker inkorrekte Einstellungen speichern. Starten Sie Bi-D Adjustment (Bidirektionale Einstellung) erneut, falls Sie den Drucker während des Einstellvorgangs ausgeschaltet haben.
Schalten Sie den Packet Mode (Paketmodus) nicht aus.
Eine Netzwerkverbindung wird nicht unterstützt.
Vergewissern Sie sich, dass der Drucker betriebsbereit ist, bevor Sie Bi-D Adjustment (Bidirektionale Einstellung) anpassen.
Falls Bi-D Adjustment (Bidirektionale Einstellung) nicht richtig gedruckt wird, stellen Sie die Schrifteinstellungen des Druckers auf die Standardwerte ein und versuchen es noch einmal.
Öffnen Sie das Menü Utility, klicken Sie dann auf das Symbol Bi-D Adjustment (Bidirektionale Einstellung). Anleitungen zum Aufrufen des Menüs Utility finden Sie unter Verwendung des Druckertreibers. Das Fenster Bi-D Adjustment (Bidirektionale Einstellung) erscheint.
Klicken Sie auf Next (Weiter).
Klicken Sie auf Print (Drucken). Der Drucker druckt drei Gruppen von Ausrichtungsmustern.
Wählen Sie die am besten ausgerichtete vertikale Linie unter den Ausrichtungsmustern. Wählen Sie dann im Fenster Bi-D Adjustment (Bidirektionale Einstellung) die entsprechende Zeilennummer.
Klicken Sie auf Set (Einstellen). Das Ausrichtungsmuster wird im Drucker gespeichert.
