Hilfe zum Benutzerhandbuch

Navigation

Dieses Handbuch enthält Verknüpfungen, um die Navigation zwischen den Abschnitten zu erleichtern. Verknüpfungen werden als farbiger, unterstrichener Text dargestellt. Wenn Sie den Mauszeiger über einer Verknüpfung bewegen, wird dieser als Handsymbol angezeigt. Klicken Sie auf die Verknüpfung, um ein verwandtes Thema oder ausführliche Informationen anzuzeigen.
Die Verknüpfungen können zudem für die Navigation im Handbuch verwendet werden:
Anfang
Kehrt zur Anfangsseite zurück.
Inhalt
Öffnet das Inhaltsverzeichnis.
Schließen Sie das Fenster, indem Sie auf die Schaltfläche  Schließen klicken.
Index
Öffnet den Index.
Schließen Sie das Fenster, indem Sie auf die Schaltfläche  Schließen klicken.
Suchen
Öffnet das Fenster Suchen, in dem Sie durch Eingabe von Suchbegriffen nach Themen suchen können.
Oben
Kehrt an den Anfang der angezeigten Seite zurück.

Ausdrucken eines Themas

Um die aktuelle Seite zu drucken, öffnen Sie das Menü File (Datei), und klicken Sie im Browser auf Print (Drucken).

Lösen von Anzeigeproblemen

Wenn eine Seite des Handbuchs nicht ordnungsgemäß angezeigt wird, klicken Sie im Browser auf die Schaltfläche Refresh (Aktualisieren) oder Reload (Neu laden).
