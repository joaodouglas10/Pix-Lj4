Принтер бағдарламасын алу

Принтер драйвері мен EPSON Status Monitor 3 қолданбасын алу
Принтер бағдарламасын қайта орнатқыңыз келсе немесе жаңартқыңыз келсе, бұрын орнатылған бағдарламаны алып тастаңыз.
Есіңізде болсын:
EPSON Status Monitor 3 қолданбасын бірнеше пайдаланушы қолданатын Windows XP немесе 2000 ортасынан алғанда, алдымен барлық клиенттердегі төте таңбаны алып тастаңыз. Таңбаны Monitoring Preferences шаршысындағы құс белгіні алу арқылы жоюға болады.

Принтер драйвері мен EPSON Status Monitor 3 қолданбасын алу

Барлық қолданбаларды тоқтатыңыз.
Windows 7, Vista немесе XP жүйелерінде:
Start (Бастау), одан кейін Control Panel (Басқару тақтасы) бөлімдерін таңдаңыз.
Windows 2000 жүйесінде:
Start (Бастау), Settings (Параметрлер) тармақтарын басып, Control Panel (Басқару тақтасы) бөлімін басыңыз.
Uninstall a program (Бағдарламаны жою) (Windows 7 немесе Vista) пәрменін басыңыз немесе Add or Remove Programs (Бағдарламаларды қосу және жою) таңбасын басыңыз (Windows XP немесе 2000).
Change or Remove Programs (Бағдарламаларды өзгерту немесе жою) (Windows XP немесе 2000 жүйелерінде ғана) пәрменін басыңыз, EPSON XXXXXX ESC/P Printer Utility Uninstall пәрменін таңдаңыз, одан кейін Uninstall/Change (Жою/Өзгерту) (Windows 7) немесе Change/Remove (Өзгерту/Алу) (Windows Vista, XP немесе 2000 жүйелерінде) пәрмендерін басыңыз.
Printer Model айдарын басыңыз, алғыңыз келетін принтердің таңбасын таңдаңыз, одан кейін OK түймесін басыңыз.
Есіңізде болсын:
Принтер үлгісі айдарынан ешнәрсенің таңдалмағанын тексеріңіз. Utility айдарын басып, EPSON Status Monitor 3(for EPSON XXXXXX ESC/P) пәрменін таңдап, OK түймесін басыңыз. EPSON Status Monitor 3 қолданбасын ғана алуға болады.
Экрандағы нұсқауларды орындаңыз.
