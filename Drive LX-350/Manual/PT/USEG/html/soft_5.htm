Utilizar o EPSON Status Monitor 3

Instalar o EPSON Status Monitor 3
Aceder ao EPSON Status Monitor 3
O EPSON Status Monitor 3 permite controlar o estado da impressora, avisa-o quando ocorre um erro na impressora e fornece instruções de resolução de problemas quando necessário.
O EPSON Status Monitor 3 está disponível quando:
A impressora está directamente ligada ao computador principal através da porta paralela [LPT1] ou da porta USB.
O sistema está configurado para suportar uma comunicação bidireccional.
O EPSON Status Monitor 3 é instalado quando se liga a impressora directamente e se instala o controlador de impressão como se descreve em Start Here ou no Manual de instalação. Se partilhar a impressora, configure o EPSON Status Monitor 3 de forma a que a impressora partilhada possa ser monitorizada no servidor de impressão e nos clientes. Consulte Instalar o EPSON Status Monitor 3 e Configurar a Impressora numa Rede.
Importante:
Embora possa imprimir com a impressora directamente ligada ao computador num local remoto, utilizando a função Remote Desktop (Ambiente de Trabalho Remoto)* do Windows 7, Vista ou XP, pode ocorrer um erro de comunicação.
* Função de ambiente de trabalho remoto: Esta função permite que o utilizador aceda a aplicações ou ficheiros existentes num computador ligado à rede do escritório, a partir de um portátil num local remoto.
Nota:
Se estiver a usar um computador com Windows 7, Vista ou XP e existirem vários utilizadores ligados ao sistema, poderá ser apresentada uma mensagem de erro de comunicação, se estiverem a ser monitorizadas várias impressoras em simultâneo.
Se o EPSON Status Monitor 3 estiver instalado com a funcionalidade de Firewall do Windows activada (recomendado), pode não conseguir monitorizar a impressora partilhada. Esse problema pode ser resolvido adicionando EEBAgent.exe a Exceptions (Excepções) na função da firewall.

Instalar o EPSON Status Monitor 3

Siga estes passos para instalar o EPSON Status Monitor 3:
No Windows 7:
Clique em Start (Iniciar) e depois em Devices and Printers (Dispositivos e Impressoras).
No Windows Vista:
Clique em Start (Iniciar), Control Panel (Painel de controlo), Hardware and Sound (Hardware e Som) e depois clique em Printers (Impressoras).
No Windows XP Professional edition:
Clique em Start (Iniciar) e depois em Printers and Faxes (Impressoras e faxes).
No Windows XP Home Edition;
Clique em Start (Iniciar), clique em Control Panel (Painel de controlo) e depois em Printers and Faxes (Impressoras e faxes).
No Windows 2000:
Clique em Start (Iniciar), seleccione Settings (Definições) e clique em Printers (Impressoras).
Clique com o botão direito do rato no ícone da impressora, clique em Printer properties (Propriedades da impressora) (Windows 7) ou Properties (Propriedades) (Windows Vista, XP e 2000) e depois clique no separador Utility (Utilitários).
Faça clique no botão Monitoring Preferences (Preferências). Será apresentada a caixa de diálogo Monitoring Preferences (Preferências).
Estão disponíveis as seguintes definições:
Select Notification (Seleccionar Aviso)
Exibe o estado do item da caixa de erro, On (Activado)/Off (Desactivado).
Active esta opção para que sejam apresentados os avisos de erro seleccionados.
Select Shortcut Icon (Selecionar Ícone de Atalho)
O ícone seleccionado é apresentado na barra de tarefas. Na parte direita do ecrã, aparece o exemplo da definição. Se fizer clique no ícone de atalho, pode aceder facilmente à caixa de diálogo Monitoring Preferences (Preferências).
Receive error notification for shared printers (Receber alertas de erro para impressoras partilhadas)
Quando esta caixa de verificação estiver seleccionada, pode receber um aviso de erro para impressoras partilhadas.
Allow monitoring of shared printer (Permitir o controlo de impressoras partilhadas)
Quando esta caixa estiver seleccionada, as impressoras partilhadas podem ser controladas por outros computadores.
Nota:
Deve aceder ao Windows como Administrador.
Nota:
Faça clique no botão Default (Predefinição) para repor as predefinições de todos os itens.

Aceder ao EPSON Status Monitor 3

Execute um dos seguintes passos para aceder ao EPSON Status Monitor 3:
Faça clique duas vezes no ícone de atalho com a forma de uma impressora que aparece na barra de tarefas. Para adicionar um ícone de atalho à barra de tarefas, seleccione o menu Utility (Utilitários) e siga as instruções apresentadas.
Abra o menu Utility (Utilitários) e faça clique no ícone EPSON Status Monitor 3. Para mais informações sobre como aceder ao menu Utility (Utilitários), consulte Utilizar o controlador de impressão.
Se aceder ao EPSON Status Monitor 3 como se descreve acima, aparece a seguinte janela de estado da impressora.
Pode visualizar as informações de estado da impressora nesta janela.
Nota:
Pode não ser possível ver o estado da impressora durante a impressão. Nesse caso, clique no botão EPSON Status Monitor 3 no separador Utility (Utilitários) e utilize a impressora mantendo aberta a janela de estado.
