Ajuda do Guia do Utilizador

Como consultar o manual

Este guia inclui hiperligações para simplificar a navegação entre as secções. As hiperligações aparecem como texto sublinhado e colorido. Quando coloca o cursor sobre uma hiperligação, este transforma-se num ícone em forma de uma mão. Clique na hiperligação para aceder a um tópico relacionado ou para obter informações detalhadas.
Pode também clicar nestas hiperligações para navegar no guia:
Página Inicial
Regressa à página de abertura.
Índice Geral
Abre o índice geral.
Para fechar a janela, clique no botão  Fechar.
Índice Remissivo
Abre o índice remissivo.
Para fechar a janela, clique no botão  Fechar.
Procurar
Abre a janela Procurar, onde pode localizar os tópicos introduzindo palavras-chave.
Topo
Regressa ao topo da página actual.

Imprimir um tópico

Para imprimir a página actual, abra o menu File (Ficheiro) e clique em Print (Imprimir) no browser.

Resolver problemas de visualização

Se uma página do guia não for apresentada correctamente, clique no botão Refresh (Actualizar) ou Reload (Recarregar) do browser.
