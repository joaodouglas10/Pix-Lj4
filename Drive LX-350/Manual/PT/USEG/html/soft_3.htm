Utilizar o controlador de impressão

Aceder ao controlador de impressão a partir do menu Iniciar
Alterar as definições do controlador de impressão
Pode aceder ao controlador de impressão a partir das aplicações do Windows ou do menu Start (Iniciar).
Quando aceder ao controlador de impressão a partir de uma aplicação do Windows, as definições efectuadas aplicam-se apenas à aplicação que estiver a utilizar. Para mais informações, consulte Aceder ao controlador de impressão a partir de aplicações do Windows.
Quando aceder ao controlador de impressão a partir do menu Start (Iniciar), as definições do controlador de impressão efectuadas aplicam-se a todas as aplicações. Para mais informações, consulte Aceder ao controlador de impressão a partir do menu Iniciar.
Consulte Alterar as definições do controlador de impressão para saber como verificar e alterar as definições do controlador de impressão.
Nota:
Ainda que muitas aplicações do Windows se sobreponham às definições efectuadas no controlador de impressão, outras não o fazem, pelo que deve verificar se as definições do controlador de impressão correspondem às suas necessidades.

Aceder ao controlador de impressão a partir de aplicações do Windows

Para aceder ao controlador de impressão a partir de uma aplicação do Windows, siga estes passos:
Seleccione Configurar impressão ou Imprimir no menu Ficheiro da aplicação. Na caixa de diálogo Print (Imprimir) ou Print Setup (Configurar impressão) apresentada, certifique-se de que o nome da impressora está seleccionado na lista pendente Name (Nome).
Clique em Impressora, Configurar, Propriedades, Opções ou Preferências. (A tecla em que faz clique depende da aplicação utilizada e poderá ter de fazer clique numa combinação destas teclas.) Será apresentada a janela Printing Preferences (Preferências de impressão) onde poderá ver os menus Layout (Esquema), Paper/Quality (Papel/Qualidade), User Defined Paper (Papel Def. Utiliz.) e Extension Settings (Def. Extensão). Estes menus contêm as definições do controlador de impressão.
Para visualizar um menu, faça clique no respectivo separador, situado na parte superior da janela. Consulte Alterar as definições do controlador de impressão para saber como alterar as definições.

Aceder ao controlador de impressão a partir do menu Iniciar

Para aceder ao controlador de impressão a partir do menu Start (Iniciar), execute as seguintes operações:
No Windows 7:
Clique em Start (Iniciar) e depois em Devices and Printers (Dispositivos e Impressoras).
No Windows Vista:
Faça clique em Start (Iniciar), Control Panel (Painel de Controlo), Hardware and Sound (Hardware e Som) e, em seguida, em Printers (Impressoras).
No Windows XP Professional edition:
Clique em Start (Iniciar) e depois em Printers and Faxes (Impressoras e faxes).
No Windows XP Home edition:
Clique em Start (Iniciar), clique em Control Panel (Painel de controlo) e depois em Printers and Faxes (Impressoras e faxes).
No Windows 2000:
Clique em Start (Iniciar), seleccione Settings (Definições) e clique em Printers (Impressoras).
Faça clique com a tecla direita do rato no ícone da impressora e faça clique em Printing Preferences (Preferências de Impressão). Será apresentada a janela Printing Preferences (Preferências de impressão) que contém os menus Layout (Esquema), Paper/Quality (Papel/Qualidade), User Defined Paper (Papel Def. Utiliz.) e Extension Settings (Def. Extensão). Estes menus contêm as definições do controlador de impressão.
Se clicar em Properties (Propriedades) (Windows Vista, XP e 2000) ou Printer properties (Propriedades da impressora) (Windows 7) no menu apresentado ao clicar com o botão direito do rato no ícone da impressora, será apresentada a janela Properties (Propriedades), que inclui menus utilizados para ajustar as definições do controlador de impressão.
Nota:
Não clique em Properties (Propriedades), embora ambas as opções Properties (Propriedades) e Printer properties (Propriedades da impressora) sejam apresentadas no menu do Windows 7.
Para visualizar um menu, faça clique no respectivo separador, situado na parte superior da janela. Para mais informações sobre como alterar as definições, consulte Descrição das Definições do Controlador de Impressão.

Alterar as definições do controlador de impressão

O seu controlador de impressão tem quatro menus que lhe permitem alterar as respectivas definições; Layout (Esquema), Paper/Quality (Papel/Qualidade), User Defined Paper (Papel Def. Utiliz.) e Extension Settings (Def. Extensão). Pode também alterar as definições no menu Utility (Utilitários) no software da impressora. Para obter uma descrição das definições disponíveis, consulte Descrição das Definições do Controlador de Impressão. Se estiver a utilizar o Windows XP ou 2000, também pode aceder à ajuda online clicando com o botão direito do rato nos itens do controlador e seleccionando What’s this? (O que é isto?).
Faça clique no botão Help (Ajuda) da janela do controlador de impressão.
Quando terminar de efectuar as definições do controlador de impressão, faça clique em OK para as aplicar ou faça clique em Cancel (Cancelar) para cancelar as alterações efectuadas.
Após ter verificado as definições do controlador de impressão e ter efectuado as alterações necessárias, poderá começar a imprimir.
