Sobre as predefinições da impressora

Modo de configuração
Alterar as predefinições
As predefinições controlam muitas das funções da impressora. Estas funções podem, geralmente, ser definidas através do software ou do controlador de impressão, no entanto, poderá ser necessário alterar uma predefinição no painel de controlo da impressora, utilizando o modo predefinido.

Page length for tractor

Esta definição permite definir o comprimento da página (em polegadas) para o papel contínuo.

Skip over perforation

Esta função está disponível apenas quando o papel contínuo está seleccionado. Se seleccionar On, a impressora deixa uma margem de uma polegada (25,4 mm) entre a última linha impressa numa página e a primeira linha impressa na página seguinte. Uma vez que as definições de margem da maioria das aplicações se sobrepõem às definições de margem especificadas no painel de controlo, deve utilizar esta definição apenas se não for possível definir as margens superior e inferior a partir da aplicação.

Auto tear off

Quando a função de corte automático estiver activada e estiver a utilizar papel contínuo com o tractor posterior, a impressora fará avançar automaticamente o picotado para a posição de corte, onde poderá cortar facilmente as páginas impressas. Quando recebe a tarefa de impressão seguinte, a impressora faz retroceder automaticamente o papel não utilizado para a posição de topo de página e começa a imprimir, de modo a que possa utilizar a totalidade da página seguinte.
A função de corte automático está disponível apenas nas seguintes condições.
Nenhuns dados de impressão são enviados durante mais de três segundos depois de introduzir uma quebra de página mantendo pressionado o botão LF/FF.
Nenhuns dados de impressão são enviados durante mais de três segundos depois de enviar um comando FF.
Quando o corte automático está desactivado, é necessário mover o picotado para a posição de corte pressionando a tecla do painel de controlo. Para mais informações, consulte Utilizar o botão Tear Off (Cortar).

Auto line feed

Quando o avanço automático de linha está activo, a impressora adiciona um código de avanço de linha a cada código de retorno do carreto (CR) recebido e muda a posição de impressão para a linha seguinte. Se a impressora adicionar uma linha extra após cada retorno do carreto, seleccione Off.

Print direction

Pode seleccionar a impressão bidireccional (Bi-D) ou a impressão unidireccional (Uni-D). Regra geral, é utilizada a definição bidireccional para uma impressão mais rápida; no entanto, a impressão unidireccional permite um alinhamento vertical mais exacto, o que é útil para a impressão de gráficos.

Software

Quando seleccionar ESC/P, a impressora funcionará no modo EPSON ESC/P. Quando seleccionar a opção IBM PPDS, a impressora emulará o funcionamento de uma impressora IBM.

0 slash

O carácter zero tem um traço () quando seleccionar. Não existe um traço (0) quando seleccionar 0. O traço permite distinguir facilmente a letra maiúscula O de um zero.

I/F (interface) mode

A impressora inclui três tipos de interface; uma interface paralela, uma de série e uma USB.
Se a impressora estiver ligada a um único computador, seleccione o modo de interface Parallel, Serial ou USB, de acordo com a interface que utiliza. Se a impressora estiver ligada a dois computadores, seleccione Auto (automático) para permitir que a impressora alterne automaticamente entre as interfaces, dependendo daquela que estiver a receber dados.

Auto I/F (interface) wait time

Esta definição só tem efeito quando o I/F mode estiver definido para Auto. Determina o tempo que a impressora aguarda quando não está a receber mais dados de uma interface, antes de verificar se está a ser enviada uma tarefa de impressão através da outra interface. Pode especificar 10 seconds ou 30 seconds para o período de espera de interface automático.

Baud rate

Esta definição determina a velocidade da transferência de dados para a interface de série. A velocidade de transmissão é definida em bits por segundo (bps). Para saber qual é a configuração adequada, consulte o manual do computador e a documentação relativa ao software.

Parity

Quando estiver seleccionado None, a verificação da paridade é desactivada. Se utilizado, o bit de paridade constitui uma forma básica de detecção de erros. Para saber qual é a configuração adequada, consulte o manual do computador e a documentação relativa ao software.

Data length

Defina o comprimento dos dados da interface de série para 7 bit ou 8 bit. Para saber qual é a configuração adequada, consulte o manual do computador e a documentação relativa ao software.

Parallel I/F bidirectional mode

O modo de transferência bidireccional da interface paralela é utilizado quando esta função está activada. Se o modo de transferência bidireccional não for necessário, desactive esta função.

Packet mode

Seleccione Auto para imprimir a partir de aplicações do Windows com o controlador de impressão incluído no CD-ROM de software fornecido com a impressora. Se ocorrerem problemas de ligação quando imprimir a partir de aplicações baseadas noutros sistemas operativos (como o DOS), desactive este modo.

Character table

Pode seleccionar uma de várias tabelas de caracteres.

International character set for Italic table

Pode seleccionar um de vários conjuntos de caracteres internacionais para a tabela de caracteres em itálico. Cada conjunto de caracteres contém oito caracteres que variam em função do país ou do idioma, para que possa personalizar a tabela de caracteres em Itálico de acordo com as suas necessidades de impressão.

Pitch

Pode seleccionar uma de seis definições de densidade.

Auto CR (retorno do carreto)

Esta definição está disponível apenas no modo de emulação IBM PPDS. Quando o retorno do carreto está activado, cada código de avanço de linha (LF) ou ESC J é acompanhado de um código de retorno do carreto. Isto move a posição de impressão para o início da linha seguinte.

Manual feed wait time

Define o tempo que a impressora aguarda entre o momento em que o papel é inserido no guia de papel e o momento em que é alimentado. Se o papel for alimentado antes de conseguir ajustar convenientemente a respectiva posição, pode aumentar o período de espera.

Buzzer (sinal sonoro)

A impressora emite um sinal sonoro quando ocorre um erro ou quando for efectuada uma operação no painel. (Consulte Utilizar os Indicadores de Erro para obter mais informações sobre os erros de impressão.) Se não pretende que a impressora emita sinais sonoros, desactive esta função.

Low-noise mode

Ao seleccionar On, o nível de ruído diminuirá, mas a velocidade de impressão será mais lenta.

IBM character table

Esta definição está disponível apenas no modo de emulação IBM PPDS. Quando seleccionar Table1, os códigos hexadecimais 80H a 90H das tabelas de caracteres serão utilizados para códigos de controlo. Quando seleccionar Table 2, os códigos hexadecimais 80H a 90H serão utilizados para caracteres.

Power off timer

Quando estiver definido um período de tempo, a impressora irá desligar após o período de tempo definido quando a mesma se encontrar no estado de suspensão. Para voltar a ligar a impressora, desligue o interruptor de alimentação e, em seguida, ligue-o novamente.

Alterar as predefinições

A tabela a seguir mostra a função básica de cada botão no modo de configuração predefinido.
Tecla
Função básica
Botões Menu (LF/FF e Load/Eject)
Permitem entrar ou sair do modo de configuração.
Item Botão
Selecciona o parâmetro seguinte do menu.
Item Botão
Selecciona o parâmetro anterior do menu.
Botão Set
Altera o valor do menu seleccionado.
Para entrar no modo de configuração e alterar as predefinições da impressora, execute as operações apresentadas em seguida:
Nota:
Antes de entrar no modo de configuração, tem de colocar folhas soltas de tamanho A4 ou Letter ou papel contínuo que tenha, no mínimo, 297 mm (11 polegadas) de comprimento e 210 mm (8,27 polegadas) de largura.
Se utilizar folhas soltas, tem de colocar uma nova folha de papel sempre que a impressora ejectar uma folha impressa.
Se desligar a impressora antes de sair do modo de configuração, qualquer alteração que tenha efectuado será cancelada e não será guardada.
Certifique-se de que a impressora tem papel.
Pressione os botões Menu (LF/FF e Load/Eject) até que a impressora emita um sinal sonoro.
A impressora entra no modo de configuração e imprime uma mensagem a pedir-lhe para seleccionar o idioma do menu de configuração. O idioma sublinhado indica a definição actual.
Nota:
Dependendo do modelo, poderá não ser possível seleccionar o idioma.
Se não estiver seleccionado o idioma que pretende, pressione o botão Item (Tear Off) até que seja indicado o idioma que pretende.
Nota:
Não é possível alterar o idioma com o botão Item (LF/FF).
Pressione o botão Set (Load/Eject) para seleccionar o idioma desejado.
A impressora imprime uma mensagem a perguntar se pretende imprimir todas as definições actuais.
Nota:
O idioma seleccionado também é utilizado quando se acede ao modo de definição de bloqueio ou ao modo de ajuste bidireccional.
Se pretender imprimir as definições actuais, pressione o botão Set. Se quiser ignorar a impressão das definições actuais, pressione o botão Item ou o botão Item.
A impressora imprime o primeiro menu e o valor actual do menu.
Pressione o botão Item ou o botão Item para seleccionar os parâmetros do menu que pretende alterar. Pressione o botão Set para percorrer os valores do parâmetro seleccionado até encontrar a definição pretendida. Depois de configurar o parâmetro seleccionado para a definição pretendida, pode alterar outros parâmetros pressionando o botão Item ou o botão Item, ou pode sair do modo de configuração pressionando os botões Menu (LF/FF e Load/Eject).
Nota:
Não pode alterar o valor da definição do item anterior. Deve pressionar o botão Set até ser seleccionado o valor que pretende.
Quando terminar todas as definições, pressione os botões Menu (LF/FF e Load/Eject).
Liberte os botões quando todas as luzes acenderem. A impressora sai do modo de configuração.
As definições efectuadas ficam guardadas como novo valor predefinido.
Nota:
Se desligar a impressora antes de sair do modo de configuração, qualquer alteração que tenha efectuado será cancelada e não será guardada.
