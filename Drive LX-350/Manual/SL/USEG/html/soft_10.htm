Odstranjevanje programske opreme za tiskalnik

Odstranjevanje gonilnika tiskalnika in pripomočka EPSON Status Monitor 3
Če želite gonilnik tiskalnika znova namestiti ali ga nadgraditi, morate odstraniti že nameščenega.
Opomba:
Če programski pripomoček EPSON Status Monitor 3 odstranjujete v večuporabniškem okolju operacijskega sistema Windows XP ali 2000, pred tem odstranite ikono bližnjice iz vseh odjemalskih računalnikov. Ikono bližnjice odstranite tako, da počistite potrditveno polje Shortcut Icon (Ikona bližnjice) v pogovornem oknu Monitoring Preferences (Možnosti spremljanja).

Odstranjevanje gonilnika tiskalnika in pripomočka EPSON Status Monitor 3

Končajte vse programe.
V operacijskem sistemu Windows 7, Vista ali XP:
Kliknite Start in nato Control Panel (Nadzorna plošča).
V operacijskem sistemu Windows 2000:
Kliknite Start, pokažite na Settings (Nastavitve) in kliknite Control Panel (Nadzorna plošča).
Kliknite Uninstall a program (Odstranitev programa) (Windows 7 ali Vista) oziroma dvokliknite ikono Add or Remove Programs (Dodaj ali odstrani programe) (Windows XP ali 2000).
Kliknite Change or Remove Programs (Spremeni ali odstrani programe) (samo Windows XP ali 2000), izberite EPSON XXXXXX ESC/P Printer Utility Uninstall in nato kliknite Uninstall/Change (Odstrani/Spremeni) (Windows 7) oziroma Change/Remove (Spremeni/Odstrani) (Windows Vista, XP ali 2000).
Kliknite kartico Printer Model (Model tiskalnika), izberite ikono tiskalnika, ki ga želite odstraniti in kliknite OK (V redu).
Opomba:
Prepričajte se, da na kartici Printer Model (Model tiskalnika) ni izbrana nobena možnost. Kliknite zavihek Utility (Pripomoček) in izberite EPSON Status Monitor 3(for EPSON XXXXXX ESC/P) in nato kliknite OK. Odstranite lahko samo EPSON Status Monitor 3.
Sledite navodilom na zaslonu.
