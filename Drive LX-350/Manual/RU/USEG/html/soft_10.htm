Удаление программного обеспечения принтера

Удаление драйвера принтера и утилиты EPSON Status Monitor 3
Если требуется установить заново или обновить драйвер принтера, то необходимо удалить установленный в данный момент драйвер принтера.
Примечание
Перед удалением утилиты EPSON Status Monitor 3 из многопользовательской среды Windows XP или 2000 удалите ярлык с каждого клиента. Для удаления ярлыка можно снять флажок опции Shortcut Icon (Значок быстрой ссылки) в диалоговом окне Monitoring Preferences (Параметры контроля).

Удаление драйвера принтера и утилиты EPSON Status Monitor 3

Закройте все приложения.
В Windows 7, Vista и XP
Нажмите кнопку Start (Пуск) и затем выберите Control Panel (Панель управления).
В Windows 2000.
Нажмите кнопку Start (Пуск), затем выберите пункты Settings (Настройка) и Control Panel (Панель управления).
Выберите пункт Uninstall a program (Удаление программы) (Windows 7 и Vista) или дважды щелкните значок Add or Remove Programs (Установка и удаление программ) (Windows XP и 2000).
Выберите пункт Change or Remove Programs (Замена или удаление программ) (только в Windows XP и 2000), выберите пункт EPSON XXXXXX ESC/P Printer Utility Uninstall, затем нажмите кнопку Uninstall/Change (Удалить/Изменить) (Windows 7) или Change/Remove (Заменить/Удалить) (Windows Vista, XP и 2000).
Выберите вкладку Printer Model (Модель принтера), выберите значок принтера, который требуется удалить, и нажмите кнопку OK.
Примечание
Убедитесь, что на вкладке Printer Model (Модель принтера) ничего не выбрано. Выберите вкладку Utility (Утилита), выберите пункт EPSON Status Monitor 3(for EPSON XXXXXX ESC/P) и нажмите кнопку OK. Вы можете удалить только утилиту EPSON Status Monitor 3.
Следуйте отображающимся на экране указаниям.
