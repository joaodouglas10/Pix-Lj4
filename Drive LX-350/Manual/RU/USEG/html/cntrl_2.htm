Режим блокировки

Изменение настроек режима блокировки
С помощью режима блокировки можно ограничить использование кнопок на панели управления.

Изменение настроек режима блокировки

Режим блокировки можно включать и отключать, а также можно изменять набор блокируемых функций в режиме настройки блокировки.
В следующей таблице перечислены основные функции каждой кнопки для режима блокировки.
Кнопка
Базовая функция
Кнопка Item
Выбор следующего пункта меню.
Кнопка Item
Выбор предыдущего пункта меню.
Кнопка Set
Изменение значения в выбранном меню.
Вы можете выбрать по своему усмотрению, какие функции принтера должны быть заблокированы при включенном режиме блокировки.
Примечание
Перед входом в режим блокировки необходимо загрузить бумагу формата А4 или Letter или непрерывную бумагу шириной не менее 210 мм (8,3 дюйма). Указания по загрузке непрерывной бумаги и отдельных листов см. в разделах Печать на непрерывной бумаге и Печать на отдельных листах.
Для вашей безопасности Epson рекомендует работать с принтером, когда крышка принтера закрыта. Принтер будет работать, если крышка поднята или снята, но максимальная безопасность достигается, когда крышка закрыта.
Чтобы изменить настройки режима блокировки, выполните указанные ниже действия.
Убедитесь, что принтер выключен.
Внимание!
После каждого выключения принтера необходимо выждать не менее пяти секунд до его повторного включения; в противном случае принтер может выйти из строя.
Включите принтер, удерживая нажатыми кнопки LF/FF и Pause.
Принтер перейдет в режим настройки блокировки, затем напечатает запрос на печать всех текущих настроек.
Чтобы распечатать текущие настройки, нажмите кнопку Set. Если требуется пропустить печать текущих настроек, нажмите кнопку Item или Item.
Принтер напечатает первое меню и текущее значение меню.
При помощи кнопок Item и Item выбирайте пункты меню, которые требуется изменить. Нажимайте кнопку Set для прокрутки значений до тех пор, пока не найдете требуемую настройку. Установив для выбранного пункта необходимое значение, можно продолжать изменять значения других пунктов при помощи кнопок Item и Item.
Выключите принтер и выйдите из режима настройки блокировки.
Примечание
Можно в любое время выйти из режима настройки блокировки, выключив принтер. Любые заданные настройки будут действовать до тех пор, пока вы снова их не измените.
