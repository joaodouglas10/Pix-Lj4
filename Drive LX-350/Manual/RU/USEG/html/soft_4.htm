Утилита настройки
В следующих таблицах перечислены настройки драйвера принтера для Windows. Обратите внимание, что набор доступных настроек зависит от используемой версии Windows. Дополнительную информацию см. в файле Help (Справка) драйвера принтера.

Настройки принтера

Настройки
Пояснения
Paper size (Формат бумаги)
Выберите формат бумаги, который требуется использовать. Если вы не видите требуемый формат бумаги, воспользуйтесь полосой прокрутки, чтобы просмотреть весь список. Если формат бумаги отсутствует в списке, см. указания в разделе Определение пользовательского формата бумаги.
Orientation (Ориентация)
Выберите пункт Portrait (Книжная) или Landscape (Альбомная).
Paper source (Источник бумаги)
Выберите источник бумаги, который требуется использовать.
Graphic (Графика) - Print Quality (Качество печати)
Выберите необходимое разрешение печати. Чем выше разрешение вы выберете, тем больше деталей будет видно на распечатке, но помните, что увеличение разрешения замедляет печать.
Halftoning (Полутона)
Выберите тип полутонов для печати графических объектов. Чтобы принтер самостоятельно определял оптимальное качество, установите значение Auto Select (Автовыбор).
Printer Font Setting (Настройка шрифта принтера)
Выберите группу кодовых таблиц символов. Можно выбрать одну из трех групп кодовых таблиц символов: Standard (Стандартная), Expanded (Расширенная) и None (Нет) (печать растрового изображения).
Если выбрана Standard (Стандартная) или Expanded (Расширенная), то принтер при печати использует шрифты устройства. Expanded (Расширенная) включает больше шрифтов, чем Standard (Стандартная).
Когда выбран пункт None (Нет), принтер вместо шрифтов устройства использует растровые изображения.
Подробные сведения о шрифтах, содержащихся в каждой из таблиц, см. в документации на принтер.
Page Order (Порядок страниц)*
Выберите порядок печати: с первой страницы до последней или с последней до первой.
Pages Per Sheet (Страниц на листе)*
Выберите количество страниц, которое должно быть напечатано на одном листе.
Copy Count (Число копий)*
Если в поле Number of copies (Число копий) выбрано более одной копии, появляется этот флажок, который позволяет выбрать, следует ли выполнять подборку страниц в копиях.
Advanced Printing Features (Дополнительные параметры печати)*
Выберите режим Enable (Включить) или Disable (Отключить) для функции Metafile Spooling (Буфер метафайлов).
Print Optimizations (Оптимизация печати)*
Выберите режим Disable (Отключить), если принтер не использует шрифты устройства, или режим Enable (Включить), если принтер использует шрифты устройства.
Printable Area (Область печати)
При выборе значения Standard (Стандарт) используется обычная область печати матричных принтеров Epson.
При выборе значения Maximum (Максимум) используется максимальная область печати.
Top Margin Priority (Приоритет верхнего поля)
Выберите, какую настройку позиции начала печати требуется использовать: настройку драйвера или настройку принтера. Дополнительную информацию см. в Справке драйвера принтера.
Packet mode (Пакетный режим)
Обычно менять эту настройку не требуется. Если вы изменили настройку Packet Mode (Пакетный режим) на принтере, необходимо соответствующим образом изменить и данную настройку. Если настройки принтера и драйвера принтера не совпадают, то при печати возможны ошибки. Дополнительную информацию см. в Справке драйвера принтера.
User Defined Paper (Пользовательская бумага)
Если в раскрывающемся списке Paper Size (Формат бумаги) нет необходимого формата бумаги, вы можете добавить его в этот список. Дополнительную информацию см. в Справке драйвера принтера.
Offset (Смещение)
Данная настройка позволяет смещать изображение на странице. То есть вы можете корректировать положение печатаемого изображения. Если при заданном значении сдвига часть изображения оказывается вне области печати принтера, то эта часть напечатана не будет.
Print Speed (Скорость печати)
Эта настройка позволяет выбирать скорость печати. Чтобы выполнять печать с использованием настроек, хранящихся в принтере, выберите пункт Printer Setting (Настройка принтера). Если выбран пункт Standard (Стандарт), печать выполняется в стандартном режиме. Если выбран пункт High Speed (Скоростная), печать выполняется в скоростном режиме.
Print direction (Направление печати)
Эта настройка позволяет включать двунаправленную печать. Если выбран пункт Printer Setting, используются настройки, хранящиеся в принтере. Если выбран пункт Bidirectional (Двунаправленная), печать осуществляется в двунаправленном режиме. Если выбран пункт Unidirectional (Однонаправленная), печать осуществляется в однонаправленном режиме.
* Более подробную информацию см. в Справке Windows.

Определение пользовательского формата бумаги

Если в раскрывающемся списке Paper Size (Формат бумаги) нет необходимого формата бумаги, вы можете добавить его в этот список.
В Windows 7.
Нажмите кнопку Start (Пуск), затем выберите пункт Devices and Printers (Устройства и принтеры).
В Windows Vista.
Нажмите кнопку Start (Пуск), затем выберите пункты Control Panel (Панель управления), Hardware and Sound (Оборудование и звук) и Printers (Принтеры).
В Windows XP Professional x64 Edition.
Нажмите кнопку Start (Пуск), затем выберите пункт Printers and Faxes (Принтеры и факсы).
В Windows XP Professional.
Нажмите кнопку Start (Пуск), затем выберите пункт Printers and Faxes (Принтеры и факсы).
В Windows XP Home Edition.
Нажмите кнопку Start (Пуск), затем выберите пункты Control Panel (Панель управления) и Printers and Faxes (Принтеры и факсы).
В Windows 2000.
Нажмите кнопку Start (Пуск), затем выберите пункты Settings (Настройка) и Printers (Принтеры).
Щелкните правой кнопкой мыши значок данного принтера, затем выберите пункт Printing Preferences (Настройка печати).
Выберите меню User Defined Paper (Пользовательская бумага).
В полях Paper Size (Формат бумаги) введите формат бумаги.
Нажмите кнопку Apply (Применить), а затем – OK.
Примечание
Дополнительную информацию см. в Справке драйвера принтера.

Настройки контроля параметров

Настройки
Пояснения
EPSON Status Monitor 3
Нажав эту кнопку, вы откроете утилиту EPSON Status Monitor 3. Чтобы контролировать работу принтера, установите флажок Monitor the printing status (Мониторинг состояния печати).
Monitoring Preferences (Параметры контроля)
Нажмите эту кнопку, чтобы открыть диалоговое окно Monitoring Preferences (Параметры контроля), которое позволяет настроить работу приложения EPSON Status Monitor 3. Дополнительную информацию см. в разделе Настройка EPSON Status Monitor 3.
