Печать внутреннего теста

Выполнение внутреннего теста принтера поможет определить источник проблемы: принтер или компьютер.
Если результаты работы внутреннего теста удовлетворительны, то принтер работает правильно и, скорее всего, ошибка связана с настройками драйвера принтера, настройками приложения, компьютером или интерфейсным кабелем. (Обязательно используйте экранированный интерфейсный кабель.)
Если печать внутреннего теста выполняется с ошибками, то проблема связана с принтером. Возможные причины и способы устранения проблем см. в разделе Проблемы и решения.
Для печати внутреннего теста можно использовать отдельные листы или непрерывную бумагу. Инструкции по загрузке бумаги см. в разделе Печать на непрерывной бумаге или Печать на отдельных листах.
Внимание!
Для выполнения внутреннего теста необходимо использовать бумагу шириной не менее 210 мм (8,27 дюйма). В противном случае головка принтера будет печатать непосредственно на валике.
Для выполнения внутреннего теста выполните указанные ниже действия.
При использовании непрерывной бумаги убедитесь, что бумага загружена, а рычаг освобождения бумаги установлен в правильную позицию. После этого выключите принтер.
Внимание!
После каждого выключения принтера необходимо выжидать не менее пяти секунд до его повторного включения; в противном случае принтер может выйти из строя.
Для выполнения теста включите принтер, удерживая кнопку LF/FF или Load/Eject.
При печати внутреннего теста используется черновой шрифт, если при включении принтера удерживается нажатой кнопка LF/FF, или качественный шрифт, если удерживается кнопка Load/Eject.
При использовании отдельных листов загрузите бумагу в принтер.
Через несколько секунд принтер автоматически загрузит бумагу и начнет печатать внутренний тест. Будет напечатана последовательность символов.
Примечание
Для временной приостановки внутреннего теста нажмите кнопку Pause. Для продолжения теста снова нажмите кнопку Pause.
Чтобы завершить выполнение внутреннего теста, нажмите кнопку Pause для остановки печати. Если в принтере осталась бумага, нажмите кнопку LF/FF для извлечения отпечатанной страницы. После этого выключите принтер.
Внимание!
Не выключайте принтер, когда он печатает внутренний тест. Для остановки печати необходимо нажимать кнопку Pause, а для извлечения отпечатанной страницы перед выключением принтера необходимо нажимать кнопку LF/FF.
