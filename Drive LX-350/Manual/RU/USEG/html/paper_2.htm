Печать на отдельных листах

Загрузка отдельных листов
В принтер можно по одному загружать отдельные листы бумаги при помощи направляющей для бумаги на принтере. Спецификации типов пригодной для использования листовой бумаги см. в разделе Бумага.
Перед печатью на одностраничных многоэкземплярных формах, конвертах и специальной бумаге обязательно устанавливайте рычаг выбора толщины бумаги в соответствующее положение. Подробные сведения см. в разделе Бумага.
Примечание.
Подробные спецификации типов пригодной для использования листовой бумаги см. в разделе Бумага.
Можно использовать направляющую для бумаги для загрузки одностраничных многоэкземплярных форм с безугольным копировальным слоем, скрепленных по верхней кромке клеевой полосой.
Вставляйте многоэкземплярные формы в направляющую для бумаги скрепленной кромкой вперед и печатной стороной вниз.

Загрузка отдельных листов

Чтобы загрузить отдельные листы, выполните указанные ниже действия.
Убедитесь, что принтер выключен. Также удостоверьтесь, что рычаг освобождения бумаги отведен в положение для отдельных листов, а направляющая для бумаги находится в вертикальном положении, как показано ниже.
Сдвиньте левую боковую направляющую в позицию 0. Затем отрегулируйте положение правой боковой направляющей в соответствии с шириной используемой бумаги.
Продвигайте лист бумаги между боковыми направляющими печатной стороной вниз, пока не почувствуете сопротивление.
Для обычного использования включите принтер, не нажимая никаких кнопок. Принтер начнет печать при получении данных.
Внимание!
Используйте рукоятку подачи бумаги c правой стороны принтера только для устранения замятия бумаги и только при выключенном принтере. В противном случае можно повредить принтер или сбить настройки позиции начала страницы.
