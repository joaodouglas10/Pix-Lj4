Использование функции Bi-D Adjustment (Настройка двунапр.)

Выполнение настроек с помощью функции Bi-D Adjustment (Настройка двунапр.)
Эта утилита служит для настройки при рассогласовании вертикальных линий.

Выполнение настроек с помощью функции Bi-D Adjustment (Настройка двунапр.)

Чтобы выполнить настройку с помощью функции Bi-D Adjustment (Настройка двунапр.), выполните указанные ниже действия.
Внимание!
Не выключайте принтер, пока он печатает настроечную таблицу. Это может привести к сохранению некорректных настроек принтера. Если в процессе настройки принтер был выключен, запустите функцию Bi-D Adjustment (Настройка двунапр.) заново.
Не отключайте функцию Packet Mode (Пакетный режим).
Сетевое подключение не поддерживается.
Прежде чем запускать функцию Bi-D Adjustment (Настройка двунапр.), убедитесь, что принтер находится в состоянии готовности.
Если настроечная таблица для функции Bi-D Adjustment (Настройка двунапр.) печатается некорректно, установите используемую по умолчанию настройку шрифта принтера и повторите попытку.
Откройте меню Utility (Утилита), затем щелкните значок Bi-D Adjustment (Настройка двунапр.). Порядок открытия меню Utility (Утилита) см. в разделе Использование драйвера принтера. Откроется окно Bi-D Adjustment (Настройка двунапр.).
Нажмите кнопку Next (Далее).
Нажмите кнопку Print (Печать). Принтер напечатает три комплекта настроечных таблиц.
Посмотрите на настроечные таблицы и найдите вертикальную линию, которая совмещена лучше других. Затем в окне Bi-D Adjustment (Настройка двунапр.) выберите соответствующий номер линии.
Нажмите кнопку Set (Установить). Настроечная таблица будет сохранена в принтере.
