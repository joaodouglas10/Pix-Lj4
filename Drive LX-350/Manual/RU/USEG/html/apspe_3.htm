Управляющие коды EPSON ESC/P

Выбор принтера EPSON ESC/P в программном обеспечении позволяет использовать расширенные функции обработки шрифтов и графики.
Общие операции:
 
ESC @, ESC U
Подача бумаги:
 
FF, LF, ESC 0, ESC 1, ESC 2, ESC 3, ESC A, CR
Формат страницы:
 
ESC (C, ESC C, ESC C 0, ESC Q, ESC l, ESC (c, ESC N, ESC O, ESC (U
Перемещение позиции печати:
 
ESC $, ESC \, ESC D, HT, ESC B, VT, ESC J
Выбор шрифта:
 
ESC k, ESC x, ESC y, ESC P, ESC M, ESC g, ESC p, ESC 4, ESC 5, ESC E, ESC F, ESC !
Расширение шрифтов:
 
ESC W, DC4, SO, DC2, SI, ESC w, ESC G, ESC H, ESC T, ESC S, ESC -
Интервалы:
 
ESC Space
Управление символами:
 
ESC t, ESC (t, ESC R, ESC %, ESC &, ESC :, ESC I, ESC 6,
ESC 7
Растровые изображения:
 
ESC K, ESC L, ESC Y, ESC Z, ESC *, ESC ^
Штрих-коды:
 
ESC (B

Управляющие коды эмуляции принтера IBM PPDS

Данный принтер поддерживает эмуляцию принтера IBM Proprinter с использованием перечисленных ниже команд. Подробную информацию см. в справочном руководстве по принтеру IBM PPDS.
Общие операции:
 
NUL, DC3, ESC j, BEL, CAN, DC1, ESC Q, ESC [ K, ESC U
Подача бумаги:
 
FF, LF, ESC 5, ESC A, ESC 0, ESC 1, ESC 2, ESC 3, CR, ESC ]
Формат страницы:
 
ESC C, ESC C0, ESC X, ESC N, ESC O, ESC 4
Перемещение позиции печати:
 
ESC d, ESC R, ESC D, HT, ESC B, VT, ESC J
Выбор шрифта:
 
DC2, ESC P, ESC :, ESC E, ESC F, ESC I, ESC [d, ESC [I
Улучшение шрифтов:
 
DC4, SO, ESC SO, ESC W, ESC [ @, SI, ESC SI, ESC G, ESC H, ESC T,
ESC S, ESC -, ESC _
Интервалы:
 
BS, SP
Управление символами:
 
ESC 6, ESC 7, ESC [ T, ESC ^, ESC ¥
Растровые изображения:
 
ESC K, ESC L, ESC Y, ESC Z
Штрих-коды:
 
ESC [f, ESC [p
Загрузка:
 
ESC =
