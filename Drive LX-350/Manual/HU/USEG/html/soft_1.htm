Operációsrendszer-verziók

Ebben a kézikönyvben az alábbi rövidítéseket használjuk.
Windows alatt a Microsoft Windows 7, Vista, XP és 2000 operációs rendszereket értjük.
Windows 7 alatt a Windows 7 Home Basic, Windows 7 Home Premium, Windows 7 Professional, Windows 7 Enterprise és Windows 7 Ultimate operációs rendszereket értjük.
A Windows Vista a Windows Vista Home Basic Edition, a Windows Vista Home Premium Edition, a Windows Vista Business Edition, a Windows Vista Enterprise Edition és a Windows Vista Ultimate Edition verziókat jelenti.
A Windows XP a következőket jelöli: Windows XP Home Edition, Windows XP Professional x64 Edition és Windows XP Professional.
A Windows 2000 a következőt jelöli: Windows 2000 Professional.
