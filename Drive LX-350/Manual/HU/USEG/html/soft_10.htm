A nyomtatószoftver eltávolítása

A nyomtatóillesztő és az EPSON Status Monitor 3 eltávolítása
Ha újra kell telepíteni vagy frissíteni kell az illesztőprogramot, akkor előbb el kell távolítani a meglévő nyomtató-illesztőprogramot.
Megjegyzés:
Az eltávolítás előtt minden ügyfélből távolítsa el a parancsikonokat az EPSON Status Monitor 3 több felhasználós Windows XP vagy 2000 környezetből történő eltávolítása során. Az ikont úgy lehet eltávolítani a Monitoring Preferences (Figyelési beállítások) párbeszédpanelből, hogy a Shortcut Icon (Parancsikon) jelölőnégyzetéből kiveszi a jelet.

A nyomtatóillesztő és az EPSON Status Monitor 3 eltávolítása

Zárjon be minden alkalmazást.
Windows 7, Vista vagy XP esetén:
Kattintson a Start gombra, majd jelölje ki a Control Panel (Vezérlőpult) elemet.
Windows 2000 esetén:
Kattintson a Start gombra, és válassza a Settings (Beállítások), majd a Control Panel (Vezérlőpult) parancsot.
Kattintson a Uninstall a program (Program eltávolítása) (Windows 7 vagy Vista) vagy kattintson kétszer a Add or Remove Programs (Programok telepítése és törlése) ikonra (Windows XP vagy 2000).
Kattintson a Change or Remove Programs (Programok módosítása vagy eltávolítása) (csak (Windows XP vagy 2000) elemre, válassza ki az EPSON XXXXXX ESC/P Printer Utility Uninstall elemet, majd kattintson az Uninstall/Change (Eltávolítás/módosítás) (Windows 7) vagy a Change/Remove (Módosítás/eltávolítás) lehetőségre (Windows Vista, XP vagy 2000).
Kattintson a Printer Model (Nyomtató típusa) lapra, jelölje ki annak a nyomtatónak az ikonját, amelyet el szeretne távolítani, majd kattintson az OK gombra.
Megjegyzés:
Győződjön meg róla, hogy semmi sincs kijelölve a Nyomtató típusa fülön. Kattintson a Utility (Segédprogram) fülre, majd válassza a EPSON Status Monitor 3(for EPSON XXXXXX ESC/P) elemet és kattintson az OK gombra. Csak az EPSON Status Monitor 3-at tudja eltávolítani.
