Informazioni sulla Guida utente

Consultazione della guida

Questa guida include collegamenti che consentono di spostarsi agevolmente tra le sezioni. I collegamenti appaiono sotto forma di testo colorato e sottolineato. Quando viene posizionato su un collegamento, il cursore assume la forma di una mano. Fare clic sul collegamento per accedere a un argomento correlato o a informazioni dettagliate.
È inoltre possibile fare clic su questi collegamenti per spostarsi nella guida:
Home
Consente di ritornare alla pagina principale.
Sommario
Apre il sommario.
È possibile chiudere la finestra facendo clic sul pulsante  Chiudi.
Indice
Apre l'indice.
È possibile chiudere la finestra facendo clic sul pulsante  Chiudi.
Ricerca
Consente di aprire la finestra Ricerca, nella quale è possibile individuare gli argomenti in base alle parole chiave immesse.
Inizio
Consente di ritornare all'inizio della pagina corrente.

Stampa di un argomento

Per stampare la pagina corrente, scegliere Print (Stampa) dal menu File del browser in uso.

Risoluzione dei problemi di visualizzazione

Se una pagina nella guida viene visualizzata in modo errato, fare clic sul pulsante Refresh (Aggiorna) o Reload (Ricarica) nel browser.
