تغيير إعدادات برنامج تشغيل الطابعة
يمكنك الوصول إلى برنامج تشغيل الطابعة من تطبيقات Windows أو من قائمة Start (ابدأ).
عند الوصول إلى برنامج تشغيل الطابعة من خلال أحد تطبيقات نظام التشغيل Windows، يتم تطبيق أية إعدادات تقوم بضبطها على التطبيق الذي تستخدمه فقط. راجع الوصول إلى برنامج تشغيل الطابعة من خلال تطبيقات نظام التشغيل Windows للحصول على المزيد من المعلومات.
عند الوصول إلى برنامج تشغيل الطابعة من قائمة Start (ابدأ)، يتم تطبيق إعدادات برنامج تشغيل الطابعة التي تقوم بضبطها على كافة التطبيقات. راجع الوصول إلى برنامج تشغيل الطابعة من القائمة Start (ابدأ) للحصول على المزيد من المعلومات.
انظر تغيير إعدادات برنامج تشغيل الطابعة للتحقق من إعدادات برنامج تشغيل الطابعة وتغييرها.
ملاحظة:
في حين يتجاهل العديد من تطبيقات نظام التشغيل Windows الإعدادات التي تم إجراؤها ببرنامج تشغيل الطابعة، فإن البعض لا يقوم بذلك، لذا يتعين عليك التحقق من أن إعدادات برنامج تشغيل الطابعة تلائم متطلباتك.

الوصول إلى برنامج تشغيل الطابعة من خلال تطبيقات نظام التشغيل Windows

اتبع هذه الخطوات للوصول إلى برنامج تشغيل الطابعة من خلال أحد تطبيقات نظام التشغيل Windows:
اختر إعداد الطباعة أو طباعة من قائمة ملف في برنامج التطبيق. وفي مربع حوار Print (طباعة) أو Print Setup (إعداد الطباعة) المعروض على الشاشة، تأكد من تحديد الطابعة الخاصة بك في القائمة Name (الاسم) المنسدلة.
انقر فوق الطابعة أو إعداد أو خصائص أو خيارات أو تفضيلات. (يعتمد الزر الذي تنقر فوقه على التطبيق، وربما قد تحتاج إلى النقر فوق مجموعة من هذه الأزرار.) وسوف يظهر إطار Printing Preferences (تفضيلات الطباعة)، وسترى قوائم Layout (التخطيط) وPaper/Quality (الورق/الجودة) وUser Defined Paper (الورق المعرّف من قبل المستخدم) وExtension Settings (إعدادات الملحق). وتحتوي تلك القوائم على إعدادات برنامج تشغيل الطابعة.
لعرض إحدى هذه القوائم، انقر فوق علامة التبويب المناظرة في أعلى الإطار. انظر تغيير إعدادات برنامج تشغيل الطابعة لتغيير الإعدادات.

الوصول إلى برنامج تشغيل الطابعة من القائمة Start (ابدأ)

اتبع هذه الخطوات للوصول إلى برنامج تشغيل الطابعة من قائمة Start (ابدأ):
فيما يخص Windows 7:
انقر Start (ابدأ)، ثم انقر Devices and Printers (الأجهزة والطابعات).
في Windows Vista:
انقر فوق Start (ابدأ) وانقر فوق Control Panel (لوحة التحكم) وانقر فوق Hardware and Sound (الأجهزة والصوت) ثم انقر فوق Printers (الطابعات).
في Windows XP Professional edition:
انقر Start (ابدأ)، ثم انقر Printers and Faxes (الطابعات والفاكسات).
في Windows XP Home edition:
انقر فوق Start (ابدأ)، وانقر فوق Control Panel (لوحة التحكم)، ثم انقر فوق Printers and Faxes (الطابعات والفاكسات).
فيما يخص Windows 2000:
انقر فوق Start (ابدأ)، ثم توجه بالمؤشر إلى Settings (إعدادات)، ثم انقر فوق Printers (طابعات).
انقر بزر الماوس الأيمن فوق أيقونة الطابعة الخاصة بك، ثم انقر فوق Printing Preferences (تفضيلات الطباعة). تعرض الشاشة إطار Printing Preferences (تفضيلات الطباعة) المحتوي على قوائم Layout (التخطيط) وPaper/Quality (الورق/الجودة) وUser Defined Paper (الورق المعرّف من قبل المستخدم) وExtension Settings (إعدادات الملحق). وتعرض هذه القوائم إعدادات برنامج تشغيل الطابعة.
عند النقر فوق Properties (خصائص) (Windows Vista وXP و2000) أو Printer properties (خصائص الطابعة) (Windows 7)في القائمة التي تظهر بالنقر بزر الماوس الأيمن فوق أيقونة الطابعة، تظهر نافذة Properties (خصائص) التي تحتوي على قوائم تستخدم في ضبط إعدادات برنامج تشغيل الطابعة.
ملاحظة:
تجنب النقر فوق Properties (خصائص) وذلك لأنه يتم عرض كل من Properties (خصائص) وPrinter properties (خصائص الطابعة) في قائمة Windows 7.
لعرض إحدى هذه القوائم، انقر فوق علامة التبويب المناظرة في أعلى الإطار. انظر نظرة عامة على إعدادات برنامج تشغيل الطابعة للحصول على معلومات حول كيفية تغيير الإعدادات.

تغيير إعدادات برنامج تشغيل الطابعة

يشتمل برنامج تشغيل الطابعة على أربع قوائم يمكنك من خلالها تغيير إعدادات برنامج تشغيل الطابعة; Layout (تخطيط) وPaper/Quality (الورق/الجودة) وUser Defined Paper (الورق المعرّف من قبل المستخدم) وExtension Settings (إعدادات الملحق). كما يمكنك تغيير الإعدادات من قائمة Utility (أداة مساعدة) الموجودة ضمن برنامج الطابعة. للحصول على نظرة عامة حول الإعدادات المتاحة، انظر نظرة عامة على إعدادات برنامج تشغيل الطابعة. إذا كنت تستخدم نظام التشغيل Windows XP أو 2000، يمكنك أيضًا الاطلاع على التعليمات الفورية بالنقر بزر الماوس الأيمن فوق العناصر الموجودة في برنامج التشغيل وتحديد What’s this؟ (ما هذا؟).
انقر فوق الزر Help في شاشة برنامج تشغيل الطابعة.
عند الانتهاء من ضبط إعدادات برنامج تشغيل الطابعة، انقر فوق OK (موافق) لتطبيق الإعدادات، أو انقر فوق Cancel (إلغاء) لإلغاء التغييرات التي أجريتها.
بمجرد مراجعة إعدادات برنامج تشغيل الطابعة، وتغييرها إذا لزم الأمر، تصبح على استعداد للطباعة.
