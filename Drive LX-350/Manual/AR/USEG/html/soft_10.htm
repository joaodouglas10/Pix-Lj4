إلغاء تثبيت برنامج الطابعة

إلغاء تثبيت برنامج تشغيل الطابعة وبرنامج EPSON Status Monitor 3
عندما ترغب في إعادة تثبيت برنامج تشغيل الطابعة أو تحديثه، قم بإلغاء تثبيت برنامج تشغيل الطابعة المثبت بالفعل.
ملاحظة:
عند إلغاء تثبيت EPSON Status Monitor 3 من بيئة Windows XP أو 2000 متعددة المستخدمين؛ قم بإزالة رمز الاختصار من كل أجهزة العملاء قبل إلغاء التثبيت. يمكنك إزالة الأيقونة بمسح مربع اختيار "رمز الاختصار" في مربع حوار Monitoring Preferences (تفضيلات المراقبة).

إلغاء تثبيت برنامج تشغيل الطابعة وبرنامج EPSON Status Monitor 3

اخرج من كافة التطبيقات.
لنظام التشغيل Windows 7 أو Vista أو XP:
انقر فوق Start (ابدأ)، ثم حدد Control Panel (لوحة التحكم).
فيما يخص Windows 2000:
انقر فوق ابدأ، ثم توجه بالمؤشر إلى إعدادات، وانقر فوق لوحة التحكم.
انقر إلغاء تثبيت برنامج (Windows 7 أو Vista) أو انقر نقرًا مزدوجًا على رمز إضافة أو إزالة برامج (Windows XP أو 2000).
انقر فوق Change or Remove Programs (تغيير البرامج أو إزالتها) (Windows XP أو 2000 فقط)، ثم حدد EPSON XXXXXX ESC/P Printer Utility Uninstall وانقر فوق Uninstall/Change (إزالة التثبيت/التغيير) (Windows 7) أو Change/Remove (تغيير/إزالة) (Windows Vista أو XP أو 2000).
انقر فوق علامة التبويب Printer Model (طراز الطابعة)، ثم اختر أيقونة الطابعة التي تريد إلغاء تثبيتها، ثم انقر فوق OK (موافق).
ملاحظة:
تأكد أنه لم يتم اختيار شيء في علامة تبويب طراز الطابعة. انقر فوق علامة التبويب Utility (أداة مساعدة) وحدد EPSON Status Monitor 3(for EPSON XXXXXX ESC/P) ثم انقر فوق OK (موافق). يمكن إلغاء تثبيت برنامج EPSON Status Monitor 3.
اتبع التعليمات التي تظهر على الشاشة.
