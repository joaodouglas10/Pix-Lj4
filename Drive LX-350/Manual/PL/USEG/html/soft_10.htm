Odinstalowywanie oprogramowania drukarki

Odinstalowywanie sterownika drukarki i programu EPSON Status Monitor 3
Jeżeli wymagana jest ponowna instalacja lub uaktualnienie sterownika drukarki, należy odinstalować dotychczasowy sterownik.
Uwaga:
W przypadku odinstalowywania programu EPSON Status Monitor 3 w środowisku wieloużytkownikowym Windows XP lub 2000, przed odinstalowaniem należy usunąć ikonę skrótu we wszystkich klientach. Ikonę można usunąć przez usunięcie zaznaczenia z pola wyboru Shortcut Icon (Ikona skrótu) w oknie dialogowym Monitoring Preferences (Preferencje monitorowania).

Odinstalowywanie sterownika drukarki i programu EPSON Status Monitor 3

Zamknij wszystkie aplikacje.
W systemie Windows 7, Vista lub XP:
Kliknij menu Start, a następnie wybierz polecenie Control Panel (Panel sterowania).
W systemie Windows 2000:
Kliknij przycisk Start, wskaż polecenie Settings (Ustawienia), a następnie kliknij pozycję Control Panel (Panel sterowania).
Kliknij pozycję Uninstall a program (Odinstaluj program) (Windows 7 lub Vista) albo kliknij dwukrotnie ikonę Add or Remove Programs (Dodaj lub usuń programy) (Windows XP lub 2000).
Kliknij pozycję Change or Remove Programs (Zmień lub usuń programy) (wyłącznieWindows XP lub 2000), wybierz pozycję EPSON XXXXXX ESC/P Printer Utility Uninstall, a następnie kliknij przycisk Uninstall/Change (Odinstaluj/Zmień) (Windows 7) lub Change/Remove (Zmień/Usuń) (Windows Vista, XP lub 2000).
Kliknij kartę Printer Model (Model drukarki), zaznacz ikonę drukarki, którą chcesz zainstalować, a następnie kliknij przycisk OK.
Uwaga:
Należy upewnić się, że niczego nie wybrano na karcie Printer Model (Model drukarki). Kliknij kartę Utility (Program narzędziowy) i wybierz pozycję EPSON Status Monitor 3(for EPSON XXXXXX ESC/P), a następnie kliknij przycisk OK. Odinstalować można tylko program EPSON Status Monitor 3.
Postępuj zgodnie z instrukcjami wyświetlanymi na ekranie.
