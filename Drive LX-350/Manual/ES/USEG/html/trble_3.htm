Corrección de problemas de USB

Asegúrese de que su ordenador es un modelo con Windows 7, Vista, XP o 2000 previamente instalado
La impresora no funciona correctamente con el cable USB
Si tiene dificultades para utilizar la impresora con una conexión USB, lea la información de este apartado.

Asegúrese de que su ordenador es un modelo con Windows 7, Vista, XP o 2000 previamente instalado

Su ordenador ha de ser un modelo con Windows 7, Vista, XP o 2000 previamente instalado. No podrá instalar ni ejecutar el controlador de impresora USB en un ordenador que no disponga de un puerto USB o que haya sido actualizado de Windows 95 a Windows Vista, XP o 2000.
Si desea más detalles sobre su ordenador, consulte a su distribuidor.

La impresora no funciona correctamente con el cable USB

Si la impresora no funciona correctamente con el cable USB, pruebe una o varias de estas soluciones:
Compruebe que el cable utilizado sea un cable USB estándar y blindado.
Si utiliza un concentrador USB, conecte la impresora a la primera fila del concentrador o directamente al puerto USB del ordenador.
