Βοήθεια για τις Οδηγίες χρήστη

Βρείτε το θέμα που θέλετε

Αυτός ο οδηγός περιέχει συνδέσεις για εύκολη μετάβαση μεταξύ των ενοτήτων. Οι συνδέσεις έχουν τη μορφή έγχρωμου και υπογραμμισμένου κειμένου. Όταν τοποθετείτε το δρομέα επάνω σε μια σύνδεση, το εικονίδιο αλλάζει και γίνεται χεράκι. Κάντε κλικ στη σύνδεση, για να μεταβείτε σε σχετικό θέμα ή σε λεπτομερείς πληροφορίες.
Μπορείτε επίσης να επιλέξετε τις παρακάτω συνδέσεις, για να περιηγηθείτε στον οδηγό:
Κεντρική σελίδα
Επιστροφή στην πρώτη σελίδα.
Περιεχόμενα
Ανοίγει τον Πίνακα περιεχομένων.
Μπορείτε να κλείσετε το παράθυρο, κάνοντας κλικ στο κουμπί  Κλείσιμο.
Ευρετήριο
Ανοίγει το Ευρετήριο.
Μπορείτε να κλείσετε το παράθυρο, κάνοντας κλικ στο κουμπί  Κλείσιμο.
Αναζήτηση
Ανοίγει το παράθυρο αναζήτησης, στο οποίο μπορείτε να εντοπίσετε θέματα πληκτρολογώντας λέξεις-κλειδιά.
Επιστροφή στην αρχή
Επιστροφή στην αρχή της τρέχουσας σελίδας.

Εκτύπωση θέματος

Για να εκτυπώσετε τη σελίδα στην οποία βρίσκεστε, ανοίξτε το μενού File (Αρχείο) και επιλέξτε το στοιχείο Print (Εκτύπωση) στο πρόγραμμα περιήγησης.

Επίλυση προβλημάτων εμφάνισης

Εάν μια σελίδα στον οδηγό δεν εμφανίζεται σωστά, κάντε κλικ στο κουμπί Refresh (Ανανέωση) ή Reload (Νέα φόρτωση) στο πρόγραμμα περιήγησης.
