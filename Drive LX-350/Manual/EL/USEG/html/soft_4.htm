Βοηθητικό πρόγραμμα ρυθμίσεων
Στους παρακάτω πίνακες, παρουσιάζονται οι ρυθμίσεις του προγράμματος οδήγησης του εκτυπωτή των Windows. Να σημειωθεί ότι δεν είναι διαθέσιμες όλες οι ρυθμίσεις σε όλες τις εκδόσεις των Windows. Για περισσότερες λεπτομέρειες, ανατρέξτε στο μενού Help (Βοήθ.) του προγράμματος οδήγησης του εκτυπωτή.

Ρυθμίσεις εκτυπωτή

Ρυθμίσεις
Επεξηγήσεις
Paper Size (Μέγ. Χαρτ.)
Επιλέξτε το μέγεθος χαρτιού που θέλετε να χρησιμοποιήσετε. Αν δεν βλέπετε το μέγεθος του χαρτιού που θέλετε, χρησιμοποιήστε τη γραμμή κύλισης για να μετακινηθείτε στη λίστα. Αν το μέγεθος του χαρτιού που θέλετε δεν περιλαμβάνεται στη λίστα, ανατρέξτε στην ενότητα Ορισμός προσαρμοσμένου μεγέθους χαρτιού για οδηγίες.
Orientation (Προσανατολισμός)
Επιλέξτε Portrait (Κατακόρυφος) ή Landscape (Οριζόντιος).
Paper Source (Προέλευση χαρτιού)
Επιλέξτε την προέλευση χαρτιού που θέλετε να χρησιμοποιήσετε.
Graphic (Γραφικό) - Print Quality (Ποιότητα εκτύπωσης)
Επιλέξτε την επιθυμητή ανάλυση εκτύπωσης. Όσο υψηλότερη είναι η ανάλυση που επιλέγετε τόσο πιο λεπτομερής θα είναι η εκτύπωση. Ωστόσο, η αύξηση της ανάλυσης μειώνει την ταχύτητα εκτύπωσης.
Halftoning (Μεσοτονικό)
Επιλέξτε τον τύπο της απόδοσης χρωμάτων που θα χρησιμοποιείται κατά την εκτύπωση γραφικών. Για να μπορέσει ο εκτυπωτής να προσδιορίσει την καλύτερη δυνατή ποιότητα ημιτονισμού, χρησιμοποιήστε το Auto Select (Αυτόματη επιλογή).
Printer Font Setting (Ρύθμιση γραμματοσειράς εκτυπωτή)
Επιλέξτε μια ομάδα πίνακα χαρακτήρων προς χρήση. Μπορείτε να επιλέξετε μία από τις τρεις ομάδες του πίνακα χαρακτήρων: Standard (Βασική), Expanded (Διευρυμένη) και None (Καμία) (Εκτύπωση Bitmap).
Όταν έχει επιλεγεί η ρύθμιση Standard (Βασική) ή Expanded (Διευρυμένη), ο εκτυπωτής χρησιμοποιεί τις γραμματοσειρές της συσκευής. Η ρύθμιση Expanded (Διευρυμένη) περιλαμβάνει περισσότερες γραμματοσειρές από τη ρύθμιση Standard (Βασική).
Όταν έχει επιλεγεί η ρύθμιση None (Καμία), ο εκτυπωτής δεν χρησιμοποιεί τις γραμματοσειρές της συσκευής, αλλά εικόνες bitmap.
Για λεπτομέρειες σχετικά με τις γραμματοσειρές που περιλαμβάνονται σε κάθε πίνακα, ανατρέξτε στο εγχειρίδιο του εκτυπωτή σας.
Page Order (Σειρά καρτελών)*
Επιλέξτε εάν η εκτύπωση θα πραγματοποιείται από μπροστά προς τα πίσω ή από πίσω προς τα μπροστά.
Pages Per Sheet (Σελίδες ανά φύλλο)*
Επιλέξτε τον αριθμό των σελίδων που θέλετε να εκτυπώσετε σε ένα μόνο φύλλο.
Copy Count (Μέτρηση αντιγράφων)*
Όταν επιλέγετε περισσότερα από ένα αντίγραφα στο πλαίσιο Number of copies, εμφανίζεται αυτό το πλαίσιο ελέγχου όπου μπορείτε να επιλέξετε εάν θα γίνει συρραφή των σελίδων.
Advanced Printing Features (Δυνατότητες σύνθετης εκτύπωσης)*
Επιλέξτε Enable (Ενεργοποίηση) ή Disable (Απενεργοποίηση) για τη λειτουργία εκτύπωσης σε ουρά μετα-αρχείων.
Print Optimizations (Βελτιστοποίηση εκτυπώσεων)*
Επιλέξτε Disable (Απενεργοποίηση) όταν ο εκτυπωτής δεν χρησιμοποιεί τις γραμματοσειρές της συσκευής και επιλέξτε Enable (Ενεργοποίηση) όταν ο εκτυπωτής χρησιμοποιεί τις γραμματοσειρές της συσκευής.
Printable Area (Περιοχή Εκτύπ.)
Αν επιλεγεί το Standard (Βασική), χρησιμοποιείται η συνηθισμένη εκτυπώσιμη περιοχή των κρουστικών εκτυπωτών Epson.
Εάν επιλέξετε Maximum (Μέγιστο), χρησιμοποιείται η μέγιστη εκτυπώσιμη περιοχή του εκτυπωτή.
Top Margin Priority (Προτ. Πάνω Περιθ.)
Επιλέξτε ποια ρύθμιση θα χρησιμοποιηθεί για τη θέση έναρξης εκτύπωσης, ρύθμιση προγράμματος οδήγησης ή ρύθμιση του εκτυπωτή. Για περισσότερες λεπτομέρειες, ανατρέξτε στο μενού Help του προγράμματος οδήγησης του εκτυπωτή.
Packet Mode (Λειτουργία πακέτου)
Συνήθως, η συγκεκριμένη ρύθμιση δεν χρειάζεται να αλλάξει. Αν αλλάξετε τις ρυθμίσεις στη λειτουργία Packet mode του εκτυπωτή, αλλάξτε αυτή τη ρύθμιση ώστε να αντιστοιχεί στις προτιμήσεις σας. Εάν οι ρυθμίσεις του εκτυπωτή και του προγράμματος οδήγησης διαφέρουν, ίσως επηρεαστεί η ποιότητα της εκτύπωσης. Για περισσότερες λεπτομέρειες, ανατρέξτε στο μενού Help του προγράμματος οδήγησης του εκτυπωτή.
User Defined Paper (Ορισμός Χαρτιού)
Αν το χαρτί που θέλετε δεν περιλαμβάνεται στην αναπτυσσόμενη λίστα Paper Size (Μέγεθος χαρτιού), μπορείτε να προσθέσετε το μέγεθος στη λίστα. Για περισσότερες λεπτομέρειες, ανατρέξτε στο μενού Help (Βοήθεια) του προγράμματος οδήγησης του εκτυπωτή.
Offset (Μετατόπιση)
Με αυτή τη ρύθμιση μπορείτε να μετατοπίσετε την εικόνα στη σελίδα σας. Επομένως, μπορείτε να προσαρμόσετε τη θέση της εκτυπωμένης εικόνας. Αν η μετατόπιση που έχετε ορίσει μετακινεί ένα τμήμα της εικόνας έξω από την εκτυπώσιμη περιοχή του εκτυπωτή, το συγκεκριμένο τμήμα δεν θα τυπωθεί.
Print Speed (Ταχύτητα Εκτύπωσης)
Η ρύθμιση αυτή σας επιτρέπει να επιλέξετε την ταχύτητα της εκτύπωσης. Επιλέξτε Printer Setting (Ρύθμιση Εκτυπωτή) όταν θέλετε να κάνετε εκτύπωση χρησιμοποιώντας τις ρυθμίσεις που είναι αποθηκευμένες στον εκτυπωτή. Εάν επιλέξετε Standard (Βασική), πραγματοποιείται εκτύπωση με βασική ταχύτητα. Εάν επιλέξετε High Speed (Υψηλή ταχύτητα), πραγματοποιείται εκτύπωση με υψηλή ταχύτητα.
Print Direction (Κατεύθυνση Εκτύπωσης)
Η ρύθμιση αυτή σας επιτρέπει να επιλέξετε εκτύπωση σε δύο κατευθύνσεις. Επιλέγοντας Printer Setting (Ρύθμιση Εκτυπωτή), χρησιμοποιείται η ρύθμιση που είναι αποθηκευμένη στον εκτυπωτή. Επιλέγοντας Bidirectional (Σε δύο κατευθύνσεις), ενεργοποιείται η λειτουργία εκτύπωσης σε δύο κατευθύνσεις. Επιλέγοντας Unidirectional (Σε μία κατεύθυνση), ενεργοποιείται η λειτουργία εκτύπωσης σε μία κατεύθυνση.
* Για λεπτομέρειες, ανατρέξτε στη βοήθεια των Windows.

Ορισμός προσαρμοσμένου μεγέθους χαρτιού

Αν το χαρτί που θέλετε δεν περιλαμβάνεται στην αναπτυσσόμενη λίστα Paper Size (Μέγ. Χαρτ.), μπορείτε να προσθέσετε το μέγεθος στη λίστα.
Για Windows 7:
Κάντε κλικ στο Start (Έναρξη) και κάντε κλικ στο Devices and Printers (Συσκευές και εκτυπωτές).
Για Windows Vista:
Κάντε κλικ στο Start (Έναρξη), στο Control Panel (Πίνακας Ελέγχου), στο Hardware and Sound (Υλικό και Ήχος) και, στη συνέχεια, στο Printers (Εκτυπωτές).
Για Windows XP Professional x64 Edition:
Κάντε κλικ στο Start (Έναρξη) και στο Printers and Faxes (Εκτυπωτές και φαξ).
Για Windows XP Professional:
Κάντε κλικ στο Start (Έναρξη) και στο Printers and Faxes (Εκτυπωτές και φαξ).
Για Windows XP Home Edition;
Κάντε κλικ στο κουμπί Start (Έναρξη), κάντε κλικ στο Control Panel (Πίνακας Ελέγχου) και, στη συνέχεια, κάντε κλικ στο Printers and Faxes (Εκτυπωτές και φαξ).
Για Windows 2000:
Κάντε κλικ στο Start (Έναρξη), Settings (Ρυθμίσεις) και κατόπιν κάντε κλικ στο Printers (Εκτυπωτές).
Κάντε δεξί κλικ στο εικονίδιο του εκτυπωτή σας και επιλέξτε Printing Preferences (Προτιμήσεις εκτύπωσης).
Επιλέξτε το μενού User Defined Paper (Χαρτί καθορισμένο από το χρήστη).
Εισάγετε το μέγεθος του χαρτιού στα πεδία Paper Size (Μέγ. Χαρτ.).
Κάντε κλικ στο Apply (Εφαρμογή) και κατόπιν στο OK.
Σημείωση:
Για περισσότερες λεπτομέρειες, ανατρέξτε στο μενού Help του προγράμματος οδήγησης του εκτυπωτή.

Επεξηγήσεις
EPSON Status Monitor 3
Για να αποκτήσετε πρόσβαση στο EPSON Status Monitor 3, κάντε κλικ σε αυτό το κουμπί. Για την εποπτεία του εκτυπωτή, βεβαιωθείτε ότι είναι επιλεγμένο το πλαίσιο ελέγχου Monitor the printing status (Εποπτεία της κατάστασης εκτύπωσης).
Monitoring Preferences (Προτιμήσεις εποπτείας)
Κάντε κλικ σε αυτό το κουμπί για να ανοίξετε το πλαίσιο διαλόγου Monitoring Preferences (Προτιμήσεις εποπτείας), όπου μπορείτε να πραγματοποιήσετε ρυθμίσεις για το EPSON Status Monitor 3. Βλ. Ρύθµιση του EPSON Status Monitor 3 για περισσότερες πληροφορίες.

Bi-D Adjustment (Προσαρμογή Εκτύπωσης σε Δύο Κατευθύνσεις)
Κάντε κλικ σε αυτό το κουμπί για να ανοίξετε το πλαίσιο διαλόγου Bi-D Adjustment (Προσαρμογή Εκτύπωσης σε Δύο Κατευθύνσεις), στο οποίο μπορείτε να διορθώσετε προβλήματα ευθυγράμμισης στις κάθετες γραμμές.
Printer Setting (Ρύθμιση Εκτυπωτή)
Κάντε κλικ σε αυτό το κουμπί για να ανοίξετε το παράθυρο διαλόγου Printer Setting (Ρύθμιση Εκτυπωτή), στο οποίο μπορείτε να αλλάξετε τις ρυθμίσεις του εκτυπωτή.
